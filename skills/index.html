📥 使用方法

  1. 复制下方任意 Skill 的安装命令
  2. 在终端粘贴运行，自动下载到 ~/.claude/skills/
  3. 下次 Claude Code 启动时自动加载，无需手动调用

安装后 Skill 根据关键词自动触发，详见各 Skill 的描述。也支持手动调用 /蒙多。
