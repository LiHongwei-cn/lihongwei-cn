macOS / Windows · 从零部署

Claude Code 部署指南
macOS · Windows 双平台

从新机到可用：安装 Claude Code → 接入 DeepSeek API → 导入全局规范 → 一键启动。
每步精确到文件路径，照做即可。

0 前提条件

你需要有 2 个东西

① DeepSeek API Key — 去 platform.deepseek.com/api_keys 创建
② 本仓库的 global-specs/ 文件夹 — GitHub 上已有，下一步 clone

最终你会得到什么

一台装了 Claude Code 的电脑，自动记住偏好（中文回复、自动 push、洁癖审查），新项目自动继承规范。

1 安装 Claude Code

安装 二选一

🍎 macOS

brew install claude-code

备选：npm install -g @anthropic-ai/claude-code

🪟 Windows

npm install -g @anthropic-ai/claude-code

备选：winget install --id Anthropic.ClaudeCode（WinGet 源）

验证安装

claude --version

输出版本号即成功。此时配置目录已自动创建（macOS ~/.claude/，Windows %USERPROFILE%\.claude\）。

2 拉取全局规范文件

Clone 仓库到桌面

🍎 macOS

cd ~/Desktop git clone https://github.com/LiHongwei-ch/lihongwei-ch.git

🪟 Windows

cd %USERPROFILE%\Desktop git clone https://github.com/LiHongwei-ch/lihongwei-ch.git

进入 global-specs 目录

🍎 macOS

cd ~/Desktop/lihongwei-ch/global-specs

🪟 Windows

cd %USERPROFILE%\Desktop\lihongwei-ch\global-specs

用 ls（macOS）/ dir（Windows）看一下，应该看到 CLAUDE.md、settings.json、rules/、skills/、memory/。

3 配置 DeepSeek API 需要改 Key

编辑 settings.json，填入真实 API Key

用文本编辑器打开 settings.json，替换 Key：

"ANTHROPIC_AUTH_TOKEN": "<你的 DeepSeek API Key>"

替换为你的真实 Key（以 sk- 开头）。其他配置（Base URL、Model）已写好。

复制到 Claude Code 配置目录

🍎 macOS

cp settings.json ~/.claude/settings.json

🪟 Windows (PowerShell)

copy settings.json %USERPROFILE%\.claude\settings.json

4 部署用户全局指令

复制 CLAUDE.md 到全局位置

🍎 macOS

cp CLAUDE.md ~/.claude/CLAUDE.md

🪟 Windows (PowerShell)

copy CLAUDE.md %USERPROFILE%\.claude\CLAUDE.md

告诉 Claude Code：你是谁、行为准则、安全红线。所有项目共享。

5 部署项目规范模板（6 份）

复制 rules/ 到全局

🍎 macOS

mkdir -p ~/.claude/rules cp rules/*.md ~/.claude/rules/

🪟 Windows (PowerShell)

mkdir %USERPROFILE%\.claude\rules copy rules\*.md %USERPROFILE%\.claude\rules\

6 份规范：agents.md、coding-style.md、git-workflow.md、performance.md、security.md、testing.md

6 安装洁癖.skill

复制 skill 文件

🍎 macOS

mkdir -p ~/.claude/skills/neat-freak/references cp skills/neat-freak/SKILL.md ~/.claude/skills/neat-freak/ cp skills/neat-freak/references/*.md ~/.claude/skills/neat-freak/references/

🪟 Windows (PowerShell)

mkdir %USERPROFILE%\.claude\skills\neat-freak\references copy skills\neat-freak\SKILL.md %USERPROFILE%\.claude\skills\neat-freak\ copy skills\neat-freak\references\*.md %USERPROFILE%\.claude\skills\neat-freak\references\

触发方式

/neat 整理一下 同步一下 收尾

Skill 自动盘点文档 → 识别变更 → 同步 → 输出摘要。

7 部署权限白名单

复制 settings.local.json

🍎 macOS

cp settings.local.json ~/.claude/settings.local.json

🪟 Windows (PowerShell)

copy settings.local.json %USERPROFILE%\.claude\settings.local.json

预授权常用命令（git add/commit/push、curl 等），避免每次都点"允许"。

此文件中部分路径为示例路径，实际使用时 Claude Code 会学习你本机的真实路径并自动更新。

8 导入跨会话记忆

最简方法

首次在项目目录启动 Claude Code，直接在对话中说需要记住的内容：

cd ~/Desktop/lihongwei-ch && claude # macOS cd %USERPROFILE%\Desktop\lihongwei-ch && claude # Windows

然后输入"记住：代码改动后自动 push 到 GitHub"，Claude Code 会自动写记忆文件到正确位置。

你也可以直接参考 global-specs/memory/ 目录中的示例记忆文件。

9 初始化项目 CLAUDE.md

为项目复制模板

🍎 macOS

cp project-CLAUDE.md ~/Desktop/lihongwei-ch/CLAUDE.md

🪟 Windows (PowerShell)

copy project-CLAUDE.md %USERPROFILE%\Desktop\lihongwei-ch\CLAUDE.md

项目级模板（MATLAB/Simulink 规范），适用于新能源汽车仿真项目。

10 验证部署

启动 Claude Code

🍎 macOS

cd ~/Desktop/lihongwei-ch claude

🪟 Windows

cd %USERPROFILE%\Desktop\lihongwei-ch claude

测试洁癖.skill

/neat

如果看到 skill 开始盘点文件、输出审查结果，说明部署成功。

测试 API 连接

随便问一个问题，如果能正常回复，说明 DeepSeek API 连接正常。

🔄 日常工作流

开始工作

进入项目目录，启动 Claude Code，直接描述问题即可。全局和项目 CLAUDE.md 已告诉 Agent 一切。

收尾存档 关键

/neat

洁癖.skill 审查文档和记忆，确保代码迭代后文档也跟上了。

新项目启动

新项目第一步永远先写 CLAUDE.md，约束先行，再写代码。

❓ 常见问题

启动 Claude Code 报 "command not found"

macOS：确认 ~/.local/bin 在 PATH 中。brew 安装通常自动配置。
Windows：确认 npm 全局安装路径在 PATH 中，或使用 npx @anthropic-ai/claude-code 启动。

API 连接失败

检查 settings.json 中 ANTHROPIC_AUTH_TOKEN 是否为有效 Key、ANTHROPIC_BASE_URL 是否为 https://api.deepseek.com/anthropic。

/neat 没反应

确认 skill 文件存在且路径正确。也可以尝试输入"整理一下"触发。
