0 是什么

Scrapling 是 Karim Shoair 开发的 Python 爬虫框架，集解析器 + Fetcher + Spider 框架于一体。测试覆盖率 92%，类型提示全覆盖，已接受数百名爬虫开发者超过一年的实战检验。

🛡️

反检测

绕过 Cloudflare Turnstile，TLS 指纹模拟，DNS 泄漏防护

🧠

自适应抓取

网站改版后自动重新定位元素，选择器不失效

⚡

极致性能

解析器比 BS4 快 ~784 倍，JSON 序列化快 10 倍

🕷️

Spider 框架

类 Scrapy async Spider，断点续抓，并发控制

🤖

AI 集成

内置 MCP 服务器，AI 辅助抓取降低 token 消耗

💻

CLI 工具

scrapling extract 终端直接抓取页面

1 安装

基础安装（仅解析器）

pip install scrapling

推荐 完整安装（全部功能）

pip install "scrapling[all]" scrapling install

包含 Fetcher、StealthyFetcher、DynamicFetcher（Playwright）、MCP 服务器、Shell 和 CLI 工具。

Docker

docker pull pyd4vinci/scrapling

预装所有浏览器依赖，生产环境推荐。

2 快速上手

▎基本请求

from scrapling.fetchers import Fetcher page = Fetcher.get('https://quotes.toscrape.com/') quotes = page.css('.quote .text::text').getall() print(quotes)

▎绕过 Cloudflare（隐秘模式）

from scrapling.fetchers import StealthyFetcher StealthyFetcher.adaptive = True page = StealthyFetcher.fetch('https://example.com', headless=True) data = page.css('.item::text').getall()

▎Spider 批量爬取

from scrapling.spiders import Spider, Request, Response class MySpider(Spider): name = "myspider" start_urls = ["https://example.com/"] concurrent_requests = 10 async def parse(self, response: Response): for item in response.css('.card'): yield { "title": item.css('h2::text').get(), "link": item.css('a::attr(href)').get(), } next_page = response.css('.pagination .next a') if next_page: yield response.follow(next_page[0].attrib['href']) result = MySpider().start() result.items.to_json("output.json")

▎CLI 快速抓取

# 抓取为 Markdown scrapling extract get 'https://example.com' content.md # 绕过 Cloudflare + 指定 CSS 选择器 scrapling extract stealthy-fetch 'https://example.com' result.html \ --css-selector '#main-content' --solve-cloudflare

3 三层 Fetcher 选择指南

Fetcher 轻量

快速 HTTP 请求，模拟浏览器 TLS 指纹和请求头，支持 HTTP/3。适合普通网页、API 接口。

Fetcher.get(url)

StealthyFetcher 反检测

高级反检测，自动绕过 Cloudflare Turnstile / Interstitial。适合受保护页面、WAF 防护站点。

StealthyFetcher.fetch(url, headless=True)

DynamicFetcher 浏览器

基于 Playwright 的完整浏览器自动化（Chromium/Google Chrome）。适合 JS 渲染页面、SPA 应用。

DynamicFetcher.fetch(url)

4 最佳实践

优先使用 Session 上下文管理器

避免资源泄漏，Session 自动管理连接池和 Cookie 持久化。

with FetcherSession(impersonate='chrome') as session: page = session.get(url)

开发阶段启用缓存回放

Spider 首次运行缓存响应到磁盘，后续直接回放，快速迭代解析逻辑。

长时爬取指定 crawldir

确保支持 Ctrl+C 暂停后从断点继续，不怕中断。

MySpider(crawldir="./crawl_data").start()

自适应选择器应对网站改版

初次抓取用 auto_save=True 保存，后续用 adaptive=True 自动重新定位。

page.css('.product', auto_save=True) # 初次 page.css('.product', adaptive=True) # 后续：网站改版后自动适配
