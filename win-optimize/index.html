2 下载与卸载

启动 WinUtil ✅ 无需梯子

右键点击 开始菜单 → Windows PowerShell (管理员) 或 终端 (管理员)，粘贴以下命令后按回车：

irm "https://christitus.com/win" | iex

工具会自动从 GitHub 下载最新版本并启动图形界面（首次加载约 10-30 秒）。

如果提示"无法加载"，先执行 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser 解除 PowerShell 限制，再重新运行。

使用 Dev 分支（可选）

想体验最新功能但可能不稳定，用以下命令：

irm "https://christitus.com/windev" | iex

如果网站打不开（备用方法）

直接从 GitHub 加载：

irm https://raw.githubusercontent.com/ChrisTitusTech/winutil/main/winutil.ps1 | iex

GitHub 需要梯子才能访问，所以这个备用方法在无梯环境下可能也失效，优先用第一条命令。

卸载 / 关闭 WinUtil ✅ 无需梯子

直接关掉窗口即可。 WinUtil 是"绿色软件"，运行时只在内存中生效，不写入系统文件、不修改注册表、不添加开机启动项。关闭窗口 = 完全卸载。

放心使用：工具不会在系统里留下任何残留，不存在"卸载不干净"的问题。

3 Windows 系统优化

这是 WinUtil 最核心的功能。打开工具后切换到 Tweaks 选项卡，你会看到一系列优化开关，按安全程度分为以下几类：

✅ 基础优化（安全，建议全开）

功能说明效果

Disable Telemetry禁用遥测收集阻止微软收集你的使用数据
Disable Activity History禁用活动历史记录微软不会记录你的操作轨迹
Disable Location Tracking禁用定位追踪保护隐私位置信息
Delete Temporary Files删除临时文件释放磁盘空间（通常几 GB）
Run Disk Cleanup运行磁盘清理进一步清理系统缓存

⚡ 进阶优化（按需开启）

功能说明推荐度
Disable Copilot禁用 Windows AI 助手推荐 用不上的可以关
Disable OneDrive禁用 OneDrive 同步推荐 不用网盘就关掉
Disable Storage Sense禁用存储感知视情况，想手动管理可关
Modern Standby Fix修复现代待机推荐 笔记本合盖后不会异常发热
Disable Edge Bloat精简 Edge 浏览器保留 Edge 但去掉多余服务

操作方法：在 Tweaks 页面勾选想开的选项 → 点击 "Run Tweaks" 执行。想撤销就再次选中该项目 → 点击 "Undo Selected Tweaks"。

4 Windows 更新控制

切换到 Updates 选项卡，可以精细控制 Windows 更新的行为，不再被强制重启或自动更新打扰。

选项说明推荐
Default 恢复 Windows 更新默认行为（微软标准推送） 重置
Security (Recommended) 仅安装安全更新，功能更新推迟 2 年，安全更新延迟 4 天推送 ✅ 强烈推荐
Disable All Updates 完全关闭 Windows 更新 ⚠️ 不推荐
Pause Updates 暂停更新一段时间 适合临时需要时用
建议选 Security (Recommended)，既能收到安全补丁保证系统安全，又不会被大型功能更新打断工作和重启。

6 自建纯净系统安装 U 盘

这是 WinUtil 最强大的功能之一 —— MicroWin（也称 Win11 Creator）。它从微软官方 ISO 出发，
在制作安装盘之前就把臃肿预装软件、遥测、硬件限制全部剔除，装出来就是干净的系统。

与传统方式的区别：传统做法是装完系统后再用工具"杀毒式"卸载预装软件，而 MicroWin 是在制作安装盘时就物理移除，装出来根本不会有这些垃圾。

准备工作 ✅ 无需梯子

① 下载微软官方 Windows ISO：从 微软官网 下载官方 Windows 11/10 镜像文件（约 5-6 GB）

② 准备一个 16GB+ 的 U 盘（用于最后制作启动盘）

③ 腾出约 15GB 磁盘空间（制作过程中需要临时缓存）

制作过程不需要梯子，但下载微软 ISO 如果慢，可以用第三方工具如 Fido 或 HeiDoc.net 加速。

启动 WinUtil 进入 MicroWin ✅ 无需梯子

管理员 PowerShell 运行：

irm "https://christitus.com/win" | iex

WinUtil 启动后，切换到 MicroWin 选项卡（旧版本可能叫 Win11 Creator）。

选择 ISO 并挂载

① 点击 "Select Windows ISO" → 选择你下载好的官方 ISO

② 点击 "Mount & Verify ISO" → 工具会自动挂载并检测可用版本

配置要移除的组件

勾选你想从安装盘中永久移除的项目：

预装 App（自动勾选）：Clipchamp、Teams、Copilot、Dev Home、OneDrive 设置、Outlook 新版、Bing 应用、纸牌等 40+

系统限制（自动勾选）：

✅ 绕过 TPM 2.0 检测（老电脑也能装 Win11）

✅ 绕过 Secure Boot 检测

✅ 绕过 CPU/内存/存储检测

✅ 启用本地账户（跳过微软账户登录）

✅ 禁用 BitLocker 和磁盘加密

✅ 禁用遥测和广告 ID

建议保留的项目：Windows Defender（杀毒软件）、计算器、截图工具、画图。保留这些不会影响纯净度，但要全部移除后自己找替代品。

开始制作

点击 "Start the process"，工具会自动：

① 解压 ISO → ② 移除勾选的组件 → ③ 清理组件存储（DISM 清理，释放 300-800 MB）→ ④ 注入自动应答文件（autounattend.xml）

整个过程约 10-30 分钟，取决于你的硬盘速度。

保存 ISO 或直写 U 盘

完成后可以选择：

① Save as ISO — 保存为自定义 ISO 文件（默认桌面，文件名 Win11_Modified_日期.iso）

② Copy to USB — 直接写入 U 盘（需提前插入）

制作启动 U 盘（推荐用 Ventoy）

如果你选择了保存 ISO，推荐用 Ventoy 制作启动盘，比 Rufus 更灵活：

① 下载 Ventoy（国内可访问）

② 插入 U 盘 → 打开 Ventoy2Disk.exe → 选择 U 盘 → 点击"安装"

③ 安装完成后，把自定义 ISO 文件直接复制到 U 盘里即可

Ventoy 的优势：一个 U 盘可以放多个 ISO，启动时选哪个装哪个，不需要反复格式化。想换系统直接把新 ISO 拖进去就行。

用 U 盘安装系统

① 插入 U 盘 → 重启电脑 → 按 F12 / F2 / Del 键进入启动菜单（不同品牌按键不同）

② 选择你的 U 盘启动 → 进入 Ventoy 菜单 → 选择自定义 ISO

③ 开始安装 → 由于硬件检测已绕过、微软账户登录已跳过，整个过程比官方版顺畅很多

装机完成后，你装出来的系统就是纯净版 —— 没有 Copilot、没有 OneDrive、没有 Xbox 垃圾、没有广告推送。开机内存占用比原版少 1-2 GB。

🍎 macOS 系统优化

虽然 macOS 本身比 Windows 干净许多（没有预装广告、注册表碎片、强制更新弹窗），但用久了也会积累垃圾文件、缓存和冗余数据。以下方法帮你保持 Mac 流畅。

Homebrew 日常维护 ✅ 免费

如果你用 Homebrew 装过软件（Git、Python 等），定期清理可以释放数 GB 空间：

brew update # 更新软件索引
brew upgrade # 升级已安装的包
brew cleanup # 删除旧版本缓存
brew autoremove # 删除无用依赖
brew doctor # 检查 Homebrew 状态

运行 brew cleanup -n 可以先预览会删除什么（-n 表示 dry-run，不实际操作）。

系统自带存储管理 ✅ 免费

macOS 内置了存储管理工具，比第三方更安全：

① 系统设置 → 通用 → 存储，等待加载完成后查看各类文件占用。

② 系统会自动推荐清理项：清空废纸篓、删除旧 Time Machine 快照、优化存储空间等。

③ Xcode 缓存（如果装了）：~/Library/Developer/Xcode/DerivedData 和 iOS DeviceSupport 文件夹常积累 10-30GB。

rm -rf ~/Library/Developer/Xcode/DerivedData/*
rm -rf ~/Library/Developer/Xcode/iOS\ DeviceSupport/*

OnyX — macOS 专用系统维护工具 ✅ 免费

OnyX 是 Mac 平台上最老牌、最受信任的免费维护工具，由法国开发者 Titanium Software 维护 20+ 年。

① 下载：titanium-software.fr/en/onyx.html（注意选对 macOS 版本，如 Sonoma / Sequoia）

② 功能：验证磁盘结构、运行系统维护脚本、清理缓存/日志/临时文件、重建 Spotlight 索引、修复权限

首次打开如果提示"无法验证开发者"，去 系统设置 → 隐私与安全性 → 仍要打开。这是正常的安全提示，不表示软件有问题。

AppCleaner — 彻底卸载应用 ✅ 免费

直接拖到废纸篓只是删了 .app，残留的配置文件、缓存、日志还留在系统里。AppCleaner 可以帮你找到并一起删掉：

① 下载：freemacsoft.net/appcleaner

② 使用方法：把要卸载的应用图标拖进 AppCleaner 窗口 → 自动列出所有关联文件 → 点"删除"

装了这个之后，每次你正常拖应用到废纸篓时，AppCleaner 会自动弹出提醒你清理残留文件。

清除系统缓存（终端命令） ⚡ 了解后再用

以下终端命令可以手动清理常见的缓存位置，适用于有一定终端使用经验的用户：

# 用户缓存（一般安全）
rm -rf ~/Library/Caches/*
# 用户日志
rm -rf ~/Library/Logs/*
# Safari 缓存
rm -rf ~/Library/Safari/LocalStorage/*
# 缩略图缓存
rm -rf ~/Library/Caches/com.apple.helpd/*
# 运行系统维护脚本（安全）
sudo periodic daily weekly monthly

sudo periodic daily weekly monthly 运行 macOS 内建的日常/每周/每月维护脚本（清理临时文件、轮转系统日志等），完全安全。

管理开机启动项 ✅ 免费

很多软件安装后会自动添加到开机启动，拖慢启动速度：

① 系统设置 → 通用 → 登录项与扩展

② 在列表中选中不需要启动的项目 → 点下方的"－"按钮移除

③ 同时检查"允许在后台运行"的分类，关闭不必要的后台项（如 Adobe 后台更新、Microsoft AutoUpdate 等）。

SSD 健康与 TRIM 检查 ✅ 免费

Mac 全部使用 SSD，定期检查健康状态可以预防数据丢失：

# 查看 SSD SMART 状态
diskutil info disk0 | grep SMART

# 确认 TRIM 已启用（Apple SSD 默认开启）
system_profiler SPSerialATADataType | grep TRIM

如果 TRIM 显示 Yes 说明正常。第三方 SSD 可能需要手动开启：sudo trimforce enable（操作前先备份）。

清理 Time Machine 本地快照 ✅ 免费

macOS 会在本地保存 Time Machine 快照，可能占用数十 GB 空间（尤其是在大文件操作后）：

# 查看本地快照
tmutil listlocalsnapshots /

# 删除所有本地快照（需要管理员密码）
sudo tmutil deletelocalsnapshots /

这些快照不影响外置备份盘上的 Time Machine 备份。删除后如果硬盘空间足够，系统会自动重建新的快照。

Apple Diagnostics 硬件诊断 ✅ 免费

怀疑硬件有问题时，Mac 自带硬件诊断工具（不需要任何软件）：

① Apple Silicon Mac：关机 → 按住电源键直到出现"选项"界面 → 按 Command + D

② Intel Mac：关机 → 开机时按住 D 键 → 选择语言后自动开始诊断

诊断完成后会显示参考代码（如 ADP000 = 无问题），可去 Apple 支持 查询具体含义。

◆ 常见问题

Q：WinUtil 安全吗？会不会有病毒？

完全开源，代码在 GitHub 上所有人可见。杀毒软件可能会报毒（因为工具需要以管理员权限修改系统设置），这是误报。如果你不放心，可以去 GitHub 查看源码确认。

Q：优化之后能还原吗？

可以。WinUtil 执行修改前会自动创建系统还原点。想撤销时，同样打开 WinUtil → Tweaks → 选中之前勾过的项目 → 点击 "Undo Selected Tweaks"。

Q：做完优化后 Windows 更新会被永久关闭吗？

不会。如果你选的是 "Security (Recommended)"，安全更新一直正常接收。如果选了 "Disable All Updates"，之后也可以随时改回 Default。

Q：我英语不好，工具界面都是英文怎么办？

界面的英文比较基础，主要是按钮和开关标签。本教程已覆盖最常用的功能，跟着步骤操作即可。你也可以用手机拍照翻译，或者试试 WinHance（有社区汉化版）。

Q：制作 MicroWin 需要什么条件？

需要从微软官网下载的官方 Windows ISO（4GB+），不建议用网上别人修改过的 ISO。制作过程需要约 15GB 临时空间，耗时 10-30 分钟。

Q：WinHance 和 WinUtil 哪个更好？

各有所长。WinUtil 功能更全（能做安装盘、装软件），WinHance 界面更友好、个性化选项更多。建议先试 WinUtil，如果觉得命令行麻烦再试 WinHance。两个都不冲突，可以同时用。

🍎 Q：Mac 用久了变慢怎么办？

三步走：① 用 OnyX 跑一遍系统维护脚本（Maintenance → Run Tasks）；② 检查存储空间，删除 Xcode 缓存和 Time Machine 本地快照；③ 用 AppCleaner 删掉不用的旧软件。通常这就能恢复流畅。如果仍不行，可能需要重置 SMC / NVRAM。

SMC 重置（解决风扇、电源、键盘背光问题）：
• Apple Silicon：关机 → 长按电源键 10 秒 → 松开，系统自动完成。
• Intel（T2 芯片）：关机 → 按住 Control+Option+Shift 7 秒 → 再加按电源键 7 秒 → 松开后开机。
NVRAM 重置（解决显示、启动盘选择问题）：仅限 Intel Mac — 关机 → 开机时按住 Option+Command+P+R 约 20 秒 → 听到两次启动声后松开。

🍎 Q：OnyX 安全吗？

OnyX 是 macOS 平台最老牌的系统维护工具，开源且由法国开发者维护 20+ 年。它本质上只是把 macOS 内置的维护脚本和命令包装成了图形界面。建议从 官网 下载，避免第三方修改版。
