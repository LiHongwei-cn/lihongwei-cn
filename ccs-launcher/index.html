Win / Mac · 一键启动

CCS 一键启动器

Claude Code 接入 DeepSeek，双击即开即用。
自动检测 CCS 代理、自动读取密钥、自动启动。

⬇ 下载启动器（Windows）

右键另存为，放到你的工作目录，双击即可

📥 下载 claude-start.bat

⬇ 下载启动器（macOS）

右键另存为，放到你的工作目录，双击即可

📥 下载 claude-desktop.command

1 工作原理

claude-start → 检测 claude 命令 → 读取环境变量密钥 → 启动 CCS 代理 → 打开 Claude Code

脚本自动完成 4 步检查：claude 已安装 → API Key 已配置 → CCS 代理在运行 → 启动对话。
任一步失败都有明确的中文提示，不需要猜哪里出了问题。
Windows 用 .bat，macOS 用 .command，功能完全一致。

2 前置条件

安装 Claude Code

需要 Node.js 18+，然后在终端运行：

npm install -g @anthropic-ai/claude-code

安装 CCS 代理

将 CCS 项目放到你的工作目录下（如 ~/ccs/ 或任意位置），确保有 server.js、index.js 或 app.js 入口文件。

CCS（Claude Code Server）是一个本地代理，将 Claude Code 的 Anthropic API 请求转发到 DeepSeek API。没有 CCS，Claude Code 无法使用 DeepSeek 模型。

配置 API Key

Windows：

setx ANTHROPIC_API_KEY "sk-你的deepseek密钥"

macOS：

echo 'export ANTHROPIC_API_KEY="sk-你的deepseek密钥"' >> ~/.zshrc && source ~/.zshrc

或设置 DEEPSEEK_API_KEY，脚本会自动读取。

3 使用方式

下载并放到工作目录

点击上方下载按钮，将启动脚本保存到你的工作目录（如桌面或项目文件夹）。

创建桌面快捷方式（可选）

Windows：右键启动脚本 → 发送到 → 桌面快捷方式。

macOS：右键启动脚本 → 制作替身 → 拖到桌面。

双击启动

Windows：双击 .bat 文件，脚本自动完成检查并在终端中打开对话界面。

macOS：首次使用需右键 .command → 打开（仅一次），之后可直接双击。

4 自定义配置

用文本编辑器打开启动脚本，修改以下变量（Win .bat / Mac .command 格式略有不同）：

:: Windows (.bat) — 修改 CCS 代理端口 set "ANTHROPIC_BASE_URL=http://127.0.0.1:3000" :: macOS (.command) — 修改 CCS 代理端口 export ANTHROPIC_BASE_URL="http://127.0.0.1:3000"
改了端口或密钥后保存文件，重新双击即可生效。

5 常见问题

双击后窗口一闪而过

这是脚本检测到错误后退出了。在 bat 所在目录打开终端，手动输入 claude-start.bat 回车，就能看到完整错误信息。

提示 claude 命令未找到

运行 npm install -g @anthropic-ai/claude-code。如果装完还是找不到，需要把 npm 全局 bin 目录加到 PATH：

:: Windows setx PATH "%PATH%;%APPDATA%\npm" # macOS echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

CCS 代理无响应

确认 CCS 已安装到工作目录。在终端手动启动测试：

cd 你的CCS目录 node server.js

如果手动能启动但脚本不能，检查启动脚本中的端口号是否一致。

连不上 DeepSeek API

CCS 代理启动了但转发出错。检查 ANTHROPIC_API_KEY 是否正确，以及是否需要代理访问外网。

:: Windows .bat 中加代理 set HTTPS_PROXY=http://127.0.0.1:7897 # macOS .command 中加代理 export HTTPS_PROXY="http://127.0.0.1:7897"

macOS 提示"无法打开，因为无法验证开发者"

这是 macOS 安全机制。右键（或 Control+点击）.command 文件 → 打开 → 点"打开"即可。只需操作一次，之后可正常双击。

Windows 打开的是黑窗口不是 Windows Terminal

在 Microsoft Store 搜索"Windows Terminal"安装（免费），再双击 bat 即可在新终端中打开。
