# PC Builder — 硬件配置推荐工具

输入预算和需求，自动推荐最优硬件搭配方案。

## 功能

- 预算滑块（3000-30000 元），实时调整
- 主板规格选择：M-ATX / ATX / ITX，实时切换
- 目标游戏多选：CS2、瓦罗兰特、三角洲行动、原神、黑神话悟空、Apex、Office、视频剪辑
- 三套推荐方案：性价比、均衡、性能优先
- 每套方案显示全部配件及单独价格
- 帧率预估：1080P / 2K 分辨率下的平均 FPS
- 性能对比柱状图：三套方案横向对比
- 升级路径建议：根据当前配置给出升级方向

## 数据来源

- 硬件价格：京东、淘宝、拼多多、抖音等国内主流平台参考价（2025 年中）
- 性能数据：基于公开评测和基准测试推算
- 价格实时变动，推荐仅供参考

## 使用方法

纯静态页面，直接打开 `index.html` 即可，无需后端。

## 技术栈

- HTML / CSS / JavaScript（无框架依赖）
- Canvas 2D 图表
- 响应式布局，移动端适配
