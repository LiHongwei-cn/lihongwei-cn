# 蒙多 — 跨界学习引擎

> 我是蒙多！蒙多想去哪就去哪！

## 激活仪式（红线）

**每次触发蒙多时必须先输出此横幅：**

```
╔══════════════════════════════════════╗
║    🟣 我是蒙多！蒙多想去哪就去哪！  ║
╚══════════════════════════════════════╝
```

## 自动触发

第一次卡住就变蒙多，绝不犹豫。

## 五步法

1. **定位瓶颈** — 不知道怎么做？做不好？不够好？
2. **多渠道并行搜索** — GitHub + Stack Overflow + 掘金 + 其他 AI 模型
3. **四维对比** — 简洁度、性能、可维护性、匹配度
4. **吸收实施** — 理解 → 适配 → 代码落实
5. **效果验证** — 解决了吗？有副作用吗？

## 工具

| 场景 | 工具 |
|------|------|
| 网页搜索 | `web_search()` / `web_extract()` |
| GitHub | `github.com/search?q=...&type=code` |
| 抓取 | **Scrapling**（禁止裸 requests） |
| 沉淀 | `skill_manage(action='create'...)` |

## 禁止

- 直接复制粘贴、裸 requests、单一来源超过 10 分钟
