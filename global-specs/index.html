2 安装

进入 global-specs/ 目录后：

macOS / Linux

运行安装脚本

chmod +x install.sh ./install.sh

Windows

双击 install.bat

或在终端中运行 install.bat

安装脚本不会覆盖已有的 settings.json 和 settings.local.json，保护现有配置。

3 配置 API Key

编辑 settings.json

安装完成后，编辑 ~/.claude/settings.json，将占位 Key 替换为真实 API Key：

{ "env": { "ANTHROPIC_AUTH_TOKEN": "sk-你的真实Key", ... } }

或通过环境变量设置：

export ANTHROPIC_AUTH_TOKEN="sk-xxx" # macOS/Linux setx ANTHROPIC_AUTH_TOKEN "sk-xxx" # Windows

4 验证安装

启动 Claude Code 并测试 Skill

claude /neat

如果 skill 开始盘点文件，说明部署成功。

5 Skills 用法

/neat — 知识库洁癖审查

收尾阶段运行，审查 CLAUDE.md、docs/、memory，确保知识库与代码同步。

/code-tidy — 代码洁癖整理

清除死代码、冗余注释、未用导入；统一排序与结构。

/蒙多 — 跨界学习引擎

遇到瓶颈时向其他 AI 和社区学习，吸收解决方案后沉淀为知识。

Skills 的详细触发条件和执行流程见各自 SKILL.md 文件。
