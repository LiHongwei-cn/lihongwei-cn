CarSim 操作：
① 双击 matlab.bat（自动 git pull + 启动 MATLAB）
② MATLAB 输入 run_carsim（生成模型 + 复制到 CarSim 目录 + 打开文件夹）
③ 在打开的文件夹里找到 CarSim.exe（红色小车图标，不是 ERD Converter）
④ CarSim → Run Control → Models 选 Simulink → 选 carsim_cruise_ctrl.slx
⑤ I/O: Export=Throttle/Brake_MPa/MotorTorque Import=Vx_kmh/Ax_g/EngineRPM
⑥ Time Step=0.001 Stop Time=30 → Run ▶
