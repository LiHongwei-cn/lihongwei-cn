3 自行部署

获取源码

项目完全开源，仓库地址：

https://github.com/LiHongwei-cn/lihongwei-cn/tree/main/bp-monitor

注册微信小程序

前往 mp.weixin.qq.com 注册个人小程序，获取 AppID 和 AppSecret。

个人主体即可，无需企业资质。服务类目选「工具 → 健康管理」。

部署后端服务

准备一台服务器（VPS / 树莓派均可），配置 HTTPS 域名：

1. 克隆仓库 → 2. 配置 .env → 3. pip install -r requirements.txt
4. uvicorn main:app → 5. Nginx 反向代理 + Let's Encrypt

配置小程序并发布

修改 app.js 中的后端地址，微信开发者工具上传代码，提交审核发布。

详细部署文档见项目仓库 README.md。
