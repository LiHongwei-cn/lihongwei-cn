CarSim-AI 高架桥仿真

基于 CarSim 2019.0 的 AI 仿真工具，模拟不同驱动类型车辆在冰雪路面上的爬坡能力。支持精确参数输入，自动生成场景和车辆配置。

🚗

立即使用

包含高架桥场景生成器、车辆参数配置器、仿真运行脚本、结果可视化工具。支持 CarSim 2019.0。

在 GitHub 浏览源码 查看使用说明
MATLAB R2016b 兼容 · CarSim 2019.0+ · 含 3 个工具函数 + 1 个示例脚本

🎬 场景描述

🏔️

燕子矶高架

模拟南京燕子矶高架桥场景，包含上坡路段、护栏、低摩擦路面

❄️

冰雪路面

下雪天路面湿滑结冰，摩擦系数 0.1-0.3，模拟真实冬季路况

在低摩擦路面上打滑，无法成功爬坡，最后滑回起点

🚙

四驱车

成功爬坡，不撞护栏，展示四驱系统在恶劣路况下的优势

🔄 工作流程

设置参数

在 MATLAB 中设置高架桥、车辆、路面参数。例如：

% 高架桥参数 params.bridge_length = 100; % 长度 [m] params.bridge_width = 8; % 宽度 [m] params.slope_angle = 15; % 坡度 [deg] params.friction = 0.2; % 摩擦系数 % 车辆参数 params.fwd_power = 100; % 前驱车功率 [kW] params.awd_power = 100; % 四驱车功率 [kW]

运行仿真

调用 run_bridge_simulation(params)，自动生成场景文件、车辆配置、仿真脚本。

CarSim 运行

在 CarSim 2019.0 中加载生成的仿真文件，运行两个仿真（前驱车和四驱车）。

分析结果

自动分析仿真结果，生成可视化图表，对比爬坡能力。

📊 仿真详解

每个工具函数的功能和参数。

🏔️
高架桥场景生成器
generate_bridge_scenario.m
生成高架桥道路、护栏、低摩擦路面场景文件，支持自定义参数。
输出：.road, .tir, .par 文件
generate_bridge_scenario(params)
🚗
车辆参数配置器
configure_vehicle.m
配置前驱/后驱/四驱车辆参数，生成 CarSim 格式的车辆配置文件。
输出：vehicle_XXX.par 文件
configure_vehicle('AWD', 100)
📊
结果可视化
visualize_results.m
可视化仿真结果，生成轨迹、速度、滑移率图表。
输出：simulation_results.png
visualize_results(params, results)
🎯
仿真主脚本
run_bridge_simulation.m
主仿真脚本，串联所有工具函数，一键运行完整仿真流程。
输出：完整仿真结果
run_bridge_simulation(params)

💻 使用示例

完整示例

以下是一个完整的仿真示例，模拟燕子矶高架场景：

%% 燕子矶高架仿真示例 % MATLAB R2016b 兼容 % 1. 设置参数 params.bridge_length = 100; % 高架桥长度 [m] params.bridge_width = 8; % 高架桥宽度 [m] params.slope_angle = 15; % 坡度角度 [deg] params.friction = 0.2; % 路面摩擦系数 params.fwd_power = 100; % 前驱车功率 [kW] params.awd_power = 100; % 四驱车功率 [kW] params.output_dir = './output';% 输出目录 % 2. 运行仿真 run_bridge_simulation(params); % 3. 查看结果 % 结果保存在 output 目录中

自定义参数

修改参数以适应不同场景：

% 更陡的坡度 params.slope_angle = 20; % 更滑的路面 params.friction = 0.15; % 更长的高架桥 params.bridge_length = 150; % 更高功率的车辆 params.fwd_power = 150; params.awd_power = 150;
