如何贡献

提交新技能

编写你的 SKILL.md，然后用提交脚本加入云仓库：

python mundo-cloud/scripts/submit_skill.py /path/to/your/SKILL.md
  • 质量评分 >= 40 分才能通过
  • 自动去重：相似度 >= 0.7 会比较质量，保留更优版本
  • 提交后自动更新 registry.json

SKILL.md 评分维度

  • 结构分 (0-30)：frontmatter、多级标题、示例代码块
  • 完整度 (0-25)：内容长度、代码块数量、表格
  • 文档分 (0-25)：注释、使用说明、注意事项
  • 新鲜度 (0-20)：版本号、更新日期

如何同步

自动同步（推荐）

配置 cron 每日自动拉取最新技能：

0 3 * * * /path/to/mundo-cloud/scripts/daily_evolve.sh

手动同步

# 拉取云端最新技能到本地
python mundo-cloud/scripts/sync_local.py

# 或者一键执行（git pull + sync + push）
bash mundo-cloud/scripts/daily_evolve.sh
