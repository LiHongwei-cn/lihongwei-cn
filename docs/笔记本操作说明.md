# 笔记本完整操作说明

## 使用前

### [笔记本] 安装 Git
1. 打开浏览器，去 https://git-scm.com/download/win
2. 下载后双击安装，一路点"下一步"就行
3. 桌面空白处右键 → 如果出现"Git Bash Here"表示安装成功

---

## 每次获取新代码

### [笔记本] ① 打开 Git Bash
- 桌面右键 → **Git Bash Here**
- 或者开始菜单搜索 "Git Bash"

### [笔记本] ② 克隆代码（仅第一次）
```
cd Desktop
git clone https://github.com/LiHongwei-ch/lihongwei-ch.git
```

### [笔记本] ③ 更新代码（以后每次）
```
cd ~/Desktop/lihongwei-ch
git pull
```

---

## 运行 MATLAB 仿真

### [笔记本] ① 启动 MATLAB R2016b

### [笔记本] ② 在 Command Window（命令窗口）逐条输入以下内容

> 下面每行开头的 `>>` 是 MATLAB 自带的，不要打
> 每输入一行按一次回车

```
cd C:\Users\你的用户名\Desktop\lihongwei-ch\matlab
```
（注意：把"你的用户名"换成实际的，比如 HP 或 Lenovo）

```
startup_setup
```
（只第一次需要，以后打开 MATLAB 运行一次即可）

### [笔记本] ③ 运行示例脚本（三选一，或全试）

**车辆动力学仿真：**
```
vehicle_dynamics
```
→ 弹出一个图，显示速度/加速度/位移三条曲线

**电机控制仿真：**
```
motor_control
```
→ 弹出一个图，显示转速/转矩/电流波形

**生成定速巡航 Simulink 模型：**
```
generate_cruise_model
```
→ 自动生成一个 Simulink 模型文件并打开

## 用完清理（还电脑给别人）

### [笔记本] ① 删除代码文件

打开 Git Bash，输入：
```
rm -rf ~/Desktop/lihongwei-ch
```

### [笔记本] ② 退出 MATLAB 授权

打开 MATLAB，在 Command Window 输入：
```
deactivate
```

### [笔记本] ③ 删除 MATLAB 历史记录

打开 Git Bash，逐条输入：
```
rm -rf ~/Documents/MATLAB
rm -rf ~/AppData/Roaming/MathWorks
```

### [笔记本] ④ 清空回收站

桌面右键回收站 → 清空回收站

### [笔记本] ⑤ 重置电脑（推荐，最彻底）

**设置 → 更新和安全 → 恢复 → 重置此电脑 → 删除所有内容**
