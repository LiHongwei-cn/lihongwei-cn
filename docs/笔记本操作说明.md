# 笔记本操作步骤

---

## 第一步：获取代码

桌面空白处右键 → **Git Bash Here**，输入：

```bash
cd Desktop
git clone https://github.com/LiHongwei-cn/lihongwei-cn.git
```

等下载完成，关掉 Git Bash。

---

## 第二步：启动 MATLAB

打开 `Desktop\lihongwei-cn\` 文件夹，**双击 `matlab.bat`**

MATLAB 会自动打开，工作目录已设到 `matlab\`。

---

## 第三步：配置路径（仅首次）

在 MATLAB 右下角的 Command Window（`>>` 提示符处）输入：

```
startup_setup
```

按回车。

---

## 第四步：运行仿真

输入以下任意命令运行：

```
vehicle_dynamics
```
（纵向动力学仿真，弹出曲线图）

```
motor_control
```
（电机 FOC 控制仿真，弹出波形图）

```
generate_cruise_model
```
（自动生成定速巡航 Simulink 模型并打开）

---

## 以后每次使用

1. **更新代码**（主电脑有新内容时）：Git Bash 输入 `cd ~/Desktop/lihongwei-cn && git pull`
2. **启动 MATLAB**：双击 `matlab.bat`
3. **运行**：直接在 Command Window 输入命令，不再需要 `cd` 和 `startup_setup`

---

## 用完还电脑

1. Git Bash：`rm -rf ~/Desktop/lihongwei-cn`
2. MATLAB Command Window：`deactivate`
3. 设置 → 重置此电脑 → 删除所有内容
