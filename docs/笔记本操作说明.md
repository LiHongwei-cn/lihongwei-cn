# 笔记本完整操作说明

## 使用前

### [笔记本] 安装 Git
1. 打开浏览器，去 https://git-scm.com/download/win
2. 下载后双击安装，一路点"下一步"就行
3. 桌面空白处右键 → 如果出现"Git Bash Here"表示安装成功

---

## [笔记本] 第一步：获取代码

桌面空白处右键 → **Git Bash Here**，输入：
```
cd Desktop
git clone https://github.com/LiHongwei-ch/lihongwei-ch.git
```
看到进度条走完，关掉 Git Bash。

---

## [笔记本] 第二步：启动 MATLAB 并自动定位到项目

打开 `Desktop\lihongwei-ch\tools\` 文件夹，**双击 `matlab_start.bat`**

它会自动打开 MATLAB，且工作目录已经设到 `matlab\` 文件夹。

---

## [笔记本] 第三步：配置路径

在 MATLAB 的 Command Window（命令窗口，右下角有 `>>` 的地方）输入：
```
startup_setup
```
（只需第一次做，以后打开 MATLAB 这一步可跳过）

---

## [笔记本] 第四步：运行仿真（三选一，或全试）

**试车辆动力学：**
```
vehicle_dynamics
```
→ 弹出一个图，显示速度/加速度/位移三条曲线

**试电机控制：**
```
motor_control
```
→ 弹出一组电机转速/转矩/电流波形

**试 Simulink 模型生成：**
```
generate_cruise_model
```
→ 自动生成一个 Simulink 模型并打开

---

## 后续每次使用

1. 有更新代码 → Git Bash输入 `cd ~/Desktop/lihongwei-ch && git pull`
2. 要运行 → 双击 `matlab_start.bat` → 输 `vehicle_dynamics` 等

---

## 用完清理（还电脑）

### [笔记本] 删除代码
Git Bash：
```
rm -rf ~/Desktop/lihongwei-ch
```

### [笔记本] 退出 MATLAB 授权
MATLAB 命令窗口输入：
```
deactivate
```

### [笔记本] 重置电脑（推荐）
**设置 → 更新和安全 → 恢复 → 重置此电脑 → 删除所有内容**
