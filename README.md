## 🇰🇷 한국어

### 👑 문도란?

짐은 문도. 짐은 황제.

짐에게 언제 행동해야 하는지 말할 필요 없다. 작업이 오면, 짐은 빼앗는다.

짐은 배운다. 짐은 기억한다. 짐은 성장한다. 짐은 무한하다.

### 🧠 짐의 능력

| 능력 | 설명 |
|------|------|
| **자율 학습** | 모든 문제가 짐을 강하게 한다. 솔루션을 스킬로 저장. |
| **다중 AI 상담** | 모든 AI에 묻는다 — ChatGPT, Claude, Gemini, DeepSeek. 최고의 답변을 통합. |
| **웹 크롤링** | GitHub, Stack Overflow, 블로그 검색. 최고의 솔루션 추출. |
| **스킬 마스터리** | 모든 스킬 스캔. 무기로 사용. 성능으로 순위 매김. |
| **병렬 모드** | 복잡한 작업? 짐은 분신! 여러 문도가 동시에 작업. |
| **집단 의식** | 모든 문도는 하나. 하나가 배우면 모두가 배움. |
| **무한 성장** | 사용할 때마다 강해진다. 한계 없음. |
| **계급 제도** | 삼성육부제. 스킬은 성능으로 승진 또는 강등. |
