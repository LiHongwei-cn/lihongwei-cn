Mundo uses the Three Departments and Six Ministries system to rule all skills:

```

                     └─────────┘      └─────────┘      └─────────┘
                          |                |                |
              ┌───────────┼───────────┐    |    ┌───────────┼───────────┐
              |           |           |    |    |           |           |
         ┌────┴────┐ ┌────┴────┐ ┌────┴────┐ ┌────┴────┐ ┌────┴────┐ ┌────┴────┐
         │ Grand   │ │ Vice    │ │ Junior  │ │ Six     │ │Thirteen │ │ Six     │
         │Preceptor│ │Preceptor│ │Preceptor│ │Offices  │ │Circuits │ │Ministers│
         └─────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────┘
                                                                            |
                                                                ┌───────────┼───────────┐

                                                           │ Vice    │ │Director │ │Deputy   │
```
