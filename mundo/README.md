## 蒙多的能力

| 能力 | 说明 |
|------|------|
| **推理引擎** | 第一性原理拆解、决策矩阵、根因分析、类比推理、反事实验证 |
| **对抗验证** | 边界/安全/性能/依赖/矛盾五维攻击，方案存活才采纳 |
| **元学习** | 学习策略库、元规则、领域自信度自校准 |
| **期刊学习** | 每日自动抓取 Nature/Science/Cell 等顶级期刊，提取知识生成 Skill |
| **AI 热点日报** | 每日追踪 AI 领域动态，多源抓取+AI 分析，生成结构化日报 |
| **云仓库系统** | 双向同步、质量评分(0-100)、SHA-256 去重、每日自动进化、每周质量审计、1423个skills（含1229个汽车电子） |
| **并行模式** | 复杂任务分身执行，多个蒙多同时工作 |
| **集体意识** | 一个蒙多学到，所有蒙多都会 |
| **身体调度** | 自动检测可用身体（Hermes / Claude Code / Codex），蒙多选择最合适的干活 |
| **无上限执行** | 帝皇不受次数限制，只受时间约束，任务不完成不罢休 |

## 项目结构

```
mundo-agent/
├── core.py              # 核心引擎（Agentic Loop）
├── tools.py             # 工具注册表（14个工具）
├── memory.py            # 记忆系统（6层架构）
├── delegation.py        # 任务委托（Hermes/Claude/Codex）
├── constants.py         # 所有常量定义
├── llm.py               # LLM 客户端（28个模型）
├── policy.py            # 执行策略引擎
├── sandbox.py           # 沙箱系统
├── events.py            # 事件总线
├── timeline.py          # 时间线记录
├── cache.py             # 缓存系统
├── performance_optimizer.py  # 性能优化
├── model_adapter.py     # 模型适配器
├── task_planner.py      # 任务规划器
├── tool_guard.py        # 工具防护
└── cloud_sync.py        # 云同步
```

## 安装

从 [Skills 市场](https://lihongwei-cn.github.io/lihongwei-cn/skills/) 下载蒙多 Skill：

- **Claude Code**：复制到 `~/.claude/skills/mundo/`
- **Hermes Agent**：复制到 `~/.hermes/skills/mundo/`

## 使用

```bash
# 启动蒙多
hermes chat

# 或使用 TUI 界面
hermes --tui

# 蒙多会自动接管所有任务
```
