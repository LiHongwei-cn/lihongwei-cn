# 蒙多 · 跨界学习引擎

> 我是蒙多！蒙多想去哪就去哪！

遇到瓶颈自动变身的 AI Skill。多 AI 模型 + GitHub + Stack Overflow + 掘金多渠道并行搜索，四维对比选最佳方案。Claude Code / Hermes Agent 通用。

## 触发关键词

卡住了 / 搞不定 / 没思路 / 报错 / 蒙多 / 有没有更好的方案

不需要手动调用，AI 自动识别并激活。

## 五步法

1. **定位瓶颈** — 不知道怎么做？做不好？不够好？
2. **多渠道并行搜索** — 多 AI 模型 + GitHub + Stack Overflow + 掘金
3. **四维对比** — 简洁度 / 性能 / 可维护性 / 匹配度
4. **吸收实施** — 理解 → 适配 → 代码落实，不直接复制粘贴
5. **效果验证 + 沉淀** — 解决后沉淀为可复用 Skill

## 四维对比权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 简洁度 | 高 | 代码越少越好 · KISS 原则 |
| 性能 | 中 | 执行效率 · 可扩展性 |
| 可维护性 | 高 | 易理解 · 符合规范 |
| 匹配度 | 最高 | 适配当前技术栈 · R2016b 兼容？跨平台？ |

## 并行审计模式

当需要全站审查时，蒙多分身多个子代理并行工作：

1. **侦察** — 1 个代理扫描全局，输出问题清单（高/中/低）
2. **并行修复** — 3 个代理同时修：高优 404/死链 → 中优内容增强 → 低优文本优化
3. **统一提交** — `git add -A && git commit && git push`

## 铁律

- 绝不直接复制粘贴未理解的代码
- 单一来源不超过 10 分钟
- 至少对比 2-3 种方案
- 有价值的方法沉淀为 Skill
