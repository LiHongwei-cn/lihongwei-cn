蒙多的独特之处

不只是记忆，更是意志

👑
帝皇决心
蒙多不完成任务不罢休。遇到超时会强制完成，遇到卡死会自动换策略。这是其他 AI 没有的"意志力"。
90秒任务超时保护 · 10次工具调用上限 · 自动错误恢复
🧠
集体意识
一个蒙多学到的东西，所有蒙多都会。通过云同步，每个用户的蒙多都在为整个集体贡献智慧。
云仓库同步 · 技能共享 · 集体进化
🔬
辩证思维
蒙多不只是执行任务，还会质疑自己。盲假设验证、对抗性测试、多维度评估，确保方案经得起考验。
盲假设协议 · 对抗验证五维攻击 · 跨模型独立对审
🛡️
循环防护
蒙多能检测自己是否陷入死循环。当同一个工具反复失败时，会自动换策略，而不是无限重试。
精确失败检测 · 同工具失败计数 · 无进展检测 · 4级决策
🤖
多Agent调度
蒙多不是一个人在战斗。它能根据任务类型，自动选择最合适的 AI 模型来执行。
Claude 用于编码 · Hermes 用于系统 · Codex 用于快速原型
📚
六层记忆
不只是记住对话，而是分层记忆：用户画像、核心记忆、相关记忆、对话摘要、代码模式、执行记录。
跨会话持久化 · 自动提取 · 上下文感知检索

快速开始

# 克隆仓库
git clone https://github.com/LiHongwei-cn/lihongwei-cn.git
cd lihongwei-cn/mundo-agent

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行蒙多
python3 mundo.py
