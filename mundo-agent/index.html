安装蒙多

1. 安装 Hermes Agent（蒙多的底层引擎）
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
2. 启动 Hermes，蒙多自动激活
hermes
3. 或手动加载蒙多
/skill mundo
4. 蒙多 CLI（独立模式）
~/.hermes/mundo-agent/mundo.sh
