前提条件

Python 3.10+
python3 --version 检查 · macOS: brew install python · Win: winget install Python.Python.3.12
Git 2.0+
git --version 检查 · macOS: brew install git · Win: winget install Git.Git
pip
随 Python 安装 · 国内加速: pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名

安装蒙多

1. 一键安装（推荐）
curl -fsSL https://raw.githubusercontent.com/LiHongwei-cn/lihongwei-cn/main/mundo-agent/install.sh | bash
2. 或手动安装
git clone https://github.com/LiHongwei-cn/lihongwei-cn.git cp -r lihongwei-cn/mundo-agent ~/.hermes/mundo-agent mkdir -p ~/bin && echo '#!/bin/bash' > ~/bin/mundo echo 'exec python3 "$HOME/.hermes/mundo-agent/mundo.py" "$@"' >> ~/bin/mundo chmod +x ~/bin/mundo
3. 启动
mundo
首次启动会引导选择 AI 模型（28 个可选）并输入 API Key。Key 仅保存在本地。
