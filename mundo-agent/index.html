0 MUNDO 是什么

146|

MUNDO 是一个独立的 AI 智能体框架，不依赖任何外部 Agent。它有自己的 LLM 客户端（30+ 个模型）、工具引擎（20 个工具）、Agentic Loop、Agent 委托、记忆系统、云同步。

5 架构

342|

MUNDO 的核心架构——42 个 Python 模块，不依赖任何外部框架。

343|
~/.hermes/mundo-agent/ 344|├── mundo.py # CLI 入口 + REPL 循环 345|├── core.py # Agentic Loop + 智能路由 + Agent 调度 346|├── llm.py # LLM 客户端（30+ provider） 347|├── tools.py # 20 个内置工具 348|├── memory.py # 记忆系统（SQLite + FTS5） 349|├── policy.py # 策略引擎 + 用户审批 350|├── security_hardening.py # 安全强化（输入验证 + 输出消毒） 351|├── delegation.py # Agent 委托（Claude/Hermes/Codex） 352|├── display.py # Rich UI（流式输出 + 状态栏） 353|├── setup.py # 首次向导（30+ 模型选择） 354|├── reflection_engine.py # 反射引擎（三思而后行） 355|├── knowledge_retriever.py # RAG 知识检索 356|├── workflow.py # 工作流引擎（任务分解） 357|├── intelligent_recovery.py # 智能错误恢复 358|├── context_mapper.py # 上下文映射 + Token 管理 359|├── model_adapter.py # 模型适配器（自动优化 prompt） 360|├── model_profiles.py # 模型画像 + 智能选择 361|├── task_planner.py # 任务规划 + 多模型协调 362|├── tool_guard.py # 工具调用守护（安全检查） 363|├── performance_optimizer.py # 性能优化（缓存 + 并行） 364|├── cache.py # 缓存管理器 365|├── sandbox.py # 沙箱执行环境 366|├── events.py # 事件总线 367|├── timeline.py # 时间线记录 368|├── build_knowledge.py # 知识库构建工具 369|├── config/ # 配置文件 370|│ ├── settings.json # 运行时配置 371|│ └── providers.json # 30+ 模型定义 372|├── knowledge.db # 知识库（SQLite） 373|├── memory.db # 记忆数据库 374|└── venv/ # Python 虚拟环境
