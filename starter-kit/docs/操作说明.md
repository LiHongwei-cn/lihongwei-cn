# 操作说明

---

## 第一次使用

### 步骤 1：获取代码

桌面空白处右键 → **Git Bash Here**，输入：

```bash
cd Desktop
git clone https://github.com/你的用户名/你的仓库名.git
```

### 步骤 2：启动 MATLAB

打开 `Desktop\你的仓库名\` 文件夹，**双击 `matlab.bat`**

### 步骤 3：配置路径（仅首次）

在 MATLAB Command Window 输入：

```
startup_setup
```

### 步骤 4：运行仿真

```
vehicle_dynamics      车辆动力学仿真
motor_control         PMSM 电机 FOC 控制仿真
generate_cruise_model 生成定速巡航 Simulink 模型
```

## 以后每次使用

1. **更新代码**：`cd ~/Desktop/你的仓库名 && git pull`
2. **启动 MATLAB**：双击 `matlab.bat`
3. **运行**：直接在 Command Window 输入命令

## 连接自己的 GitHub

```bash
cd ~/Desktop/你的仓库名
git remote set-url origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

## 用 AI 生成新代码

配合 Claude Code 或 ChatGPT：描述你的仿真需求 → AI 生成 MATLAB 脚本 → 保存到 `matlab/examples/` → 提交到 GitHub → 笔记本上运行。
