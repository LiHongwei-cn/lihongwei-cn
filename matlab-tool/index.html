1 第一次使用（笔记本）

获取代码

桌面空白处右键 → Git Bash Here，输入：

cd Desktop
git clone https://github.com/LiHongwei-ch/lihongwei-ch.git

等下载完成，关掉 Git Bash。

启动 MATLAB

打开 Desktop\lihongwei-ch\ 文件夹，双击 matlab.bat

MATLAB 会自动打开，工作目录已设到 matlab\。

配置路径（仅首次）

在 MATLAB Command Window（>> 提示符处）输入：

startup_setup

按回车，只需做一次。

运行仿真

输入以下任意命令运行：

% 车辆动力学仿真，弹出曲线图
vehicle_dynamics

% PMSM 电机 FOC 控制仿真，弹出波形图
motor_control

% 自动生成定速巡航 Simulink 模型并打开
generate_cruise_model

2 以后每次使用

更新代码

主电脑有新内容时，在 Git Bash 输入：

cd ~/Desktop/lihongwei-ch && git pull

启动并运行

双击 matlab.bat → 直接在 Command Window 输入命令

不再需要 cd 和 startup_setup，工作目录和路径已自动配置。

3 用完还电脑

清理痕迹

# 删除项目文件
rm -rf ~/Desktop/lihongwei-ch

% MATLAB 中注销许可证
deactivate

然后 设置 → 重置此电脑 → 删除所有内容

学校电脑注意清理个人文件，不要遗留代码或账户信息。
