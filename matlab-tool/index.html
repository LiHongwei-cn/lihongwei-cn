1 从零开始：在你的电脑上搭环境

以下步骤适用于刚重装系统、什么软件都没有的新电脑。

安装 Git

Git 是用来下载代码的工具。打开浏览器访问 git-scm.com，下载安装包，一路点"下一步"完成安装。

安装后在桌面空白处右键，菜单里出现 Git Bash Here 就说明装好了。

下载并解压安装包

点击本页上方 「下载安装包（ZIP）」 按钮，下载后解压到桌面。

注册 GitHub 并创建仓库

打开 github.com 注册账号。然后点击右上角 + 号 → New repository，输入仓库名称，勾选 Public，点击 Create repository。

找到你的 GitHub 用户名和仓库名

关联远程仓库之前，需要知道两个信息：

① 用户名：打开 github.com 并登录 → 点击右上角你的头像 → 看弹出卡片第一行，Signed in as 后面的就是你的用户名（一串英文字母，不是邮箱）。

② 仓库名：回到 github.com 首页 → 点左上角头像回到个人主页 → 找到你刚才创建的仓库 → 点进去 → 浏览器地址栏显示的就是：

github.com/你的用户名/你的仓库名

把地址栏里的用户名和仓库名记下来，下一步要用。

例如地址栏是 github.com/zhangsan/matlab-project，用户名就是 zhangsan，仓库名就是 matlab-project。

关联你的远程仓库

在解压后的文件夹里右键 → Git Bash Here，把下面命令中的中文替换成你上一步找到的用户名和仓库名，然后执行：

git remote set-url origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main

第一条命令把本地项目和你的 GitHub 仓库绑定。
第二条命令把代码上传到 GitHub。

验证是否成功：刷新你的 GitHub 仓库页面，能看到文件列表就说明成功了。

2 在笔记本（或任何电脑）上运行

安装 MATLAB

如果你用的是公用电脑，MATLAB 可能已经装好了。如果没有，安装 MATLAB R2016b 或更新版本即可。

获取代码

桌面空白处右键 → Git Bash Here，输入：

cd Desktop
git clone https://github.com/你的用户名/你的仓库名.git

启动 MATLAB

打开 Desktop\你的仓库名\ 文件夹，双击 matlab.bat

脚本会自动 git pull 获取最新代码，然后启动 MATLAB，工作目录自动设到 matlab/。

每次双击都会先拉取最新代码，不需要手动输 git 命令。如果有新代码推送到 GitHub，双击 bat 后自动同步。

配置路径（仅首次需要）

在 MATLAB 右下角 Command Window（>> 提示符处）输入：

startup_setup

运行仿真

% 车辆动力学仿真，弹出曲线图
vehicle_dynamics

% PMSM 电机 FOC 控制，弹出波形图
motor_control

% 生成定速巡航 Simulink 模型并打开
generate_cruise_model

3 CarSim-Simulink 联合仿真

将 Simulink 控制器接入 CarSim 2019.0 整车模型，实现真正的高精度联合仿真。

配置环境（仅首次）

在 MATLAB Command Window 输入：

sim_setup

脚本自动检测 CarSim 安装路径，添加 Simulink 接口到 MATLAB 搜索路径。

生成联合仿真模型

在 MATLAB Command Window 输入：

build_carsim_model

自动生成 carsim_cruise_ctrl.slx，包含速度控制器（PI）和 CarSim I/O 接口。

模型包含完整的速度闭环：目标车速 → PI 控制器 → 节气门/制动 → CarSim 车辆 → 实际车速反馈

CarSim 中配置 I/O 通道

打开 CarSim 2019.0 → Run Control → Models 下拉选 Simulink → 浏览选择刚生成的 carsim_cruise_ctrl.slx

点击 I/O Channels，按脚本打印的映射表配置（共 3 个 Import + 3 个 Export）。

Step Size 设为 0.001 s，与模型一致。

运行联合仿真

CarSim 中点击 Run → CarSim 自动调用 Simulink 模型 → 车辆动力学由 CarSim 计算，控制器由 Simulink 计算。

结果可在 CarSim 中查看动画和曲线，也可在 Simulink Scope 中查看信号。

自定义控制器

打开 carsim_cruise_ctrl.slx，修改 PI 参数（双击 Kp、Ki 模块）。

如需更复杂的控制策略（FOC 电机控制、能量管理），在模型中添加子系统，保持 Import/Outport 端口号不变即可。

Simulink 模型文件在 matlab/carsim/ 目录下，通过 git 自动同步到所有电脑。

4 用 AI 生成新代码

描述需求

用自然语言告诉 AI 你想要什么仿真，例如："写一个电池 SOC 估算的 MATLAB 代码，用安时积分法"

保存脚本

将 AI 生成的代码保存到 matlab/examples/ 目录下。

提交并同步到 GitHub

git add .
git commit -m "添加电池SOC估算仿真"
git push

笔记本上运行

在笔记本上打开 Git Bash，git pull 拉取新代码，然后双击 matlab.bat 启动 MATLAB 运行。

只需新增 .m 文件，不用修改任何配置。startup_setup 会自动添加路径，新文件可以直接运行。
