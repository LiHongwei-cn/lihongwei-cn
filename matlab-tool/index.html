1 初始化（首次使用）

下载并解压

点击上方按钮下载 ZIP，解压到你想要的位置。

创建 GitHub 仓库

在 github.com/new 创建一个空仓库，复制仓库地址。

关联远程仓库

在解压后的目录里打开 Git Bash，执行：

git remote set-url origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main

2 在笔记本上运行

获取代码

桌面空白处右键 → Git Bash Here，输入：

cd Desktop
git clone https://github.com/你的用户名/你的仓库名.git

启动 MATLAB

打开 Desktop\你的仓库名\ 文件夹，双击 matlab.bat

MATLAB 自动打开，工作目录自动设好。

配置路径（仅首次）

在 Command Window 输入：

startup_setup

运行仿真

% 车辆动力学仿真
vehicle_dynamics

% PMSM 电机 FOC 控制
motor_control

% 生成定速巡航 Simulink 模型
generate_cruise_model

3 用 AI 生成新代码

描述需求

用自然语言告诉 AI 你想要什么仿真，例如："写一个电池 SOC 估算的 MATLAB 代码，用安时积分法"

保存脚本

将 AI 生成的代码保存到 matlab/examples/ 目录下。

提交并同步

git add .
git commit -m "添加电池SOC估算仿真"
git push

笔记本上运行

git pull 拉取新代码，在 MATLAB 中运行即可。

只需新增 .m 文件，不用修改任何配置。startup_setup 会自动添加路径。
