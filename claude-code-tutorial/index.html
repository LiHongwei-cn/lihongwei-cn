0 Claude Code 是什么

Claude Code 是 Anthropic 公司推出的终端版 AI Agent 框架。它不像 ChatGPT 那样在网页上聊天，而是在你的终端（命令行）里工作，可以直接读/写你电脑上的文件、执行命令、管理项目。

🤖

Agent 框架

不是网页聊天，是在终端里帮你写代码、改文件、执行命令的智能助手

🔌

模型自由

可以接入 Claude / DeepSeek / GLM / MiniMax 等任意模型，不用被绑定

🛡️

国内可用

框架本身不封号，搭配国产模型不需要魔法、不需要外国手机号

📁

本地运行

代码在你自己电脑上执行，不经过第三方服务器，安全可控

核心概念：Agent 框架 = 身体，模型 = 脑子。Claude Code 是身体，你可以给它装上国产模型的脑子（如 DeepSeek），效果依然很好。这也是它不会被封号的原因。

1 安装 Claude Code

▎Mac 安装

有魔法 ✅ 无魔法（Homebrew）⭐

打开终端 ✅ 无需魔法

在 Mac 上，按 Command + 空格 搜索「终端」（Terminal），打开终端 App。

你会看到一个黑底白字（或白底黑字）的窗口，末尾有 $ 符号在闪烁，这就是命令行。

✅ 有魔法 用官方命令安装

在终端中粘贴以下命令，按下回车：

curl -fsSL https://claude.ai/install.sh | bash

① 脚本会自动下载并安装 Claude Code，等待几秒到几十秒

② 看到 "Claude Code installed successfully" 就装好了

③ 如果提示 "请将 ~/.local/bin 加入 PATH"，按终端给出的 echo 'export PATH=...' 那行命令复制执行即可

④ 输入 claude --version 检查，有版本号输出表示成功

⭐ 推荐 ✅ 无魔法 用 Homebrew 安装

如果没魔法，先安装 Homebrew（Mac 的包管理器）：

步骤 A — 安装 Homebrew：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

① 粘贴后回车，会提示按 Return/回车 继续 → 按回车

② 等待几分钟（取决于网速），看到 "Installation successful" 完成

步骤 B — 把 Homebrew 加入 PATH：

安装完成后，终端里会显示几行 echo 命令，复制那几行粘贴到终端执行。完成后输入 brew --version 验证。

步骤 C — 安装 Claude Code：

brew install --cask claude-code@latest

安装速度略慢，等待完成即可。完成后终端输入 claude 看到小螃蟹 🦀 图标即成功。

注意如果复制命令到终端报错，可能是因为公众号编辑器修改了格式，把命令手动敲一遍或用 Claude 修正后再粘贴。

▎Windows 安装

有魔法 ✅ 无魔法（WinGet）⭐

先安装 Git ✅ 无需魔法

Claude Code 在 Windows 上依赖 Git Bash 来执行命令，所以必须先把 Git 装上。

① 在任务栏搜索 "终端"（Terminal）或按 Win + R 输入 wt 打开 Windows 终端

② 粘贴以下命令回车,安装时"不开魔法"速度更快：

winget install Git.Git

③ 等待安装完成，显示 "已成功安装" 即可。

已经装过 Git 的可以跳过这步。在终端输入 git --version 有版本号就说明有了。

✅ 有魔法 用官方命令安装

确保 Git 装好后，在终端粘贴以下命令回车：

irm https://claude.ai/install.ps1 | iex

等待安装完成，输入 claude --version 验证。

⭐ 推荐 ✅ 无魔法 用 WinGet 安装

如果没魔法，直接用 Windows 自带的包管理器 WinGet：

winget install Anthropic.ClaudeCode

安装成功后，终端输入 claude 能看到小螃蟹 🦀 图标表示成功。

此时框架装好了，但还没接模型脑子，下一步就来配置模型。

2 接入 DeepSeek 模型（国产，免费额度）

Claude Code 装好之后只是一个空壳，还需要接上模型才能用。如果有 Claude 账号，直接用 claude 命令登录即可（本文不展开）。
这里教大家接 DeepSeek（国产模型），不需要 Claude 账号、不需要魔法、不需要外国手机号。以下两种方法二选一。

为什么选 DeepSeek？国产模型中性价比最高，有免费额度，API 国内可以直接访问。模型效果稳定，对 Claude Code 兼容性好。
需要先做一件事：去 platform.deepseek.com 注册账号 → 控制台 → API Keys → 创建新 Key → 复制保存。

▎方法一（推荐⭐）：直接配置环境变量

不需要装任何额外工具，直接修改 Claude Code 的配置文件即可。这是最简单、最稳定的方式。

找到配置文件位置 ✅ 无需魔法

在电脑上找到 Claude Code 的配置文件：

Mac：打开终端 → open ~/.claude/settings.json

Windows：在文件管理器地址栏输入 %USERPROFILE%\.claude\settings.json 回车

如果这个文件还不存在，创建一个新文件命名为 settings.json 放到 .claude 文件夹里。

写入 DeepSeek 配置 ✅ 无需魔法

用记事本/文本编辑打开 settings.json，粘贴以下内容：

{ "env": { "ANTHROPIC_BASE_URL": "https://api.deepseek.com/anthropic", "ANTHROPIC_AUTH_TOKEN": "sk-你的deepseek_api_key", "ANTHROPIC_MODEL": "deepseek-v4-pro", "ANTHROPIC_DEFAULT_HAIKU_MODEL": "deepseek-v4-pro", "ANTHROPIC_DEFAULT_SONNET_MODEL": "deepseek-v4-pro", "ANTHROPIC_DEFAULT_OPUS_MODEL": "deepseek-v4-pro" }, "theme": "dark" }

记得把 sk-你的deepseek_api_key 替换成你在 DeepSeek 平台创建的真正 API Key。

这六行环境变量的意思是：把 Claude Code 内部调用的所有模型（Haiku/Sonnet/Opus）都指向 DeepSeek 的接口。这样无论 Claude Code 请求哪个模型，实际走的都是 DeepSeek。

验证配置 ✅ 无需魔法

保存文件后，在终端输入 claude 启动 Claude Code。

如果能正常进入对话界面（看到小螃蟹 🦀 和 > 输入框），说明 DeepSeek 接入成功。

如果提示 API Key 错误，回 DeepSeek 平台检查你的 Key 是否正确、账户是否有余额/免费额度。

▎方法二（备选）：CC Switch 图形化工具

如果不想手动编辑配置文件，也可以用 CC Switch 这个图形化工具来管理模型切换（GitHub 下载，需魔法）。

安装 CC Switch

Mac：终端依次执行：

brew tap farion1231/ccswitch
brew install --cask cc-switch

Windows：下载安装包双击运行，一路 Next 完成安装。

配置 DeepSeek

打开 CC Switch → 在 Claude Code 栏下点右上角 "+" → 选 DeepSeek 或自定义：

- API Key：填你的 DeepSeek API Key

- Base URL：填 https://api.deepseek.com/anthropic

- 模型名：填 deepseek-v4-pro

- 其他选项保持默认 → 点击 "添加" → 自动切换到此模型

之后在 Claude Code 中输入 /model 即可切换模型。

3 启动 Claude Code

首次启动 ✅ 无需魔法

回到终端，输入 claude 回车。

首次启动会经过几个设置步骤：

① 选择颜色主题：出现暗色/亮色两个选项，用键盘上下键选择，回车确认。以后可在 Claude Code 里输入 /theme 修改。

② 安全提示：显示两行注意事项——Claude 会犯错，代码要过眼再放行；只在你信任的目录中使用。直接回车继续。

③ 终端设置：问是否启用 Claude Code 推荐终端设置（快捷键换行 + 视觉提示）。选"是"（推荐）。

④ 信任目录：问是否信任当前目录。选"是"。

确认进入对话界面 ✅ 无需魔法

以上设置完成后，终端中出现小螃蟹 🦀 图标和一个输入框（> 提示符），

这意味着你已经成功启动了搭载 DeepSeek 的 Claude Code，可以开始对话了！

后续要切换模型：在 CC Switch 中配置好其他模型 → 在 Claude Code 中输入 /model 即可切换。

⭐ 推荐 建议的启动方式

以后每次使用，建议先 cd 到你想要工作的文件夹，再启动 Claude Code：

① 打开终端，输入：

cd 你的文件夹路径

Mac 技巧：输入 cd （注意后面有空格），然后把文件夹从 Finder 拖进终端窗口，路径会自动填入。

② 然后在当前目录下启动：

claude --dangerously-skip-permissions

这个 --dangerously-skip-permissions 参数会跳过每次操作前的确认提示，省去反复点 allow 的麻烦。

为什么要 cd 到文件夹再启动？Claude Code 默认只读取当前目录下的文件，这样上下文更专注、更聪明，也避免污染其他项目。

4 设置 CLAUDE.md（使用规范）

这是用好 Claude Code 最重要的一步。CLAUDE.md 是一份约束文件，告诉 Claude Code 你是谁、想要怎么合作、有什么规则。
它分为两层：全局级（所有项目通用）和项目级（单个项目专用）。

全局 CLAUDE.md（所有项目通用）

放在 ~/.claude/CLAUDE.md，只要打开 Claude Code 就会自动加载。文件不要超过 80 行，最多最多不超过 200 行。

## 关于我 我的名字 / 身份 / 职业背景，非程序员的话一定要写出来 我用 Claude Code 做 [具体用途 1] 和 [具体用途 2] ## 思维原则 所有决策从问题本质出发，不因「惯例如此」照搬。 不要夸我的想法好、不要问「这是个好问题吗」、直接给结论。 给我真实判断，方案有问题直接指出。发现更好的做法直接说。 ## 约束先行 新项目先写 CLAUDE.md，新目录先定结构约定。 已有规范的项目严格遵守其 CLAUDE.md 中的约定。 ## 沟通方式 - 默认中文，代码、命令、变量名用英文 - 结论先行，再给理由 - 遇到模糊需求，先给最合理方案，再问要不要调整 ## 自主边界（红线，必须先问我） - 删除文件、目录或 git 历史 - 修改 .env、密钥、token - git push、git rebase、git reset --hard - 安装全局依赖或修改系统配置 ## 工程纪律 - 改完主动验证，不要只改不验 - 不要注释掉报错来绕过，找根本原因 - 密钥、token 不进代码、不进 commit

项目级 CLAUDE.md

放在每个项目的根目录下（项目目录/CLAUDE.md），只在该项目中生效。

这个文件不需要你自己写，先 cd 到项目目录启动 Claude Code，然后直接跟它聊你的项目需求、你关心的规范，让它帮你生成一份 CLAUDE.md 即可。

举个例子：如果你有一个 code/my 文件夹用来放各种实验小项目，在里面放一个项目级 CLAUDE.md，告诉它"如果是一个新需求，自动创建子文件夹存放"，以后你只管提需求，文件管理都交给它。

6 搭建 Telegram AI 机器人

把 AI 接入 Telegram，直接发要求给机器人，自动识别意图：代码生成 → 推送到 GitHub，论文报告 → 保存到 docs。笔记本上双击 matlab.bat 即可运行最新代码。

准备工作

你需要提前准备好以下三项：

① 一台电脑（能长期运行，或使用云服务器）

② DeepSeek API Key：去 platform.deepseek.com 注册 → 创建 API Key

③ Telegram Bot Token：Telegram 中搜索 @BotFather → 发送 /newbot → 按提示取名 → 获得一串 token（格式如 123456:ABC-DEF...）

安装 Python ✅ 无需魔法

此机器人是 Python 写的，需要 Python 3.8 以上版本。

Windows：打开 python.org 下载安装包 → 安装时务必勾选"Add Python to PATH" → 完成安装。

Mac：终端输入 python3 --version，如果没装则去 python.org 下载。

安装后在终端输入 python --version 或 python3 --version 验证。

安装依赖库 ✅ 无需魔法

终端执行以下命令，安装机器人所需的 Python 包：

pip install python-telegram-bot openai

如果提示 pip 不存在，试试 pip3 install python-telegram-bot openai

创建机器人脚本 ✅ 无需魔法

新建一个文件，命名为 tgbot.py，完整代码可在仓库 bot/tgbot.py 中获取。
以下为简化版，展示自动识别 + 代码生成推送的核心逻辑：

import os from telegram import Update from telegram.ext import ApplicationBuilder, MessageHandler, filters, ContextTypes from openai import OpenAI TELEGRAM_TOKEN = os.environ["TELEGRAM_TOKEN"] # 从环境变量读取，绝不硬编码 DEEPSEEK_API_KEY = os.environ["DEEPSEEK_API_KEY"] client = OpenAI(api_key=DEEPSEEK_API_KEY, base_url="https://api.deepseek.com") SYSTEM_PROMPT = """回复原则：极度简洁，代码直给，不废话。默认中文。""" async def handle_message(update, context): text = update.message.text response = client.chat.completions.create( model="deepseek-v4-pro", messages=[ {"role": "system", "content": SYSTEM_PROMPT}, {"role": "user", "content": text} ] ) await update.message.reply_text(response.choices[0].message.content) app = ApplicationBuilder().token(TELEGRAM_TOKEN).build() app.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, handle_message)) app.run_polling()
完整版见 GitHub 仓库 bot/tgbot.py，基于 DeepSeek API 直接回复，支持文字和图片消息，无需手动输入命令。

运行机器人 ✅ 无需魔法

在终端中 cd 到 tgbot.py 所在的目录，执行：

python tgbot.py

看到 "Bot 已启动，等待消息..." 即表示运行成功。

此时去 Telegram 找到你的机器人（搜索你之前 @BotFather 给它取的名字），发一条消息测试。

直接对话 ⭐ 核心功能

机器人基于 DeepSeek API，直接回复所有消息：

MATLAB/Simulink 代码：直接回复可运行代码，兼容 R2016b

论文/报告：生成正文和分析内容

图片分析：拍照发作业、实验数据图，直接解读

普通聊天：正常对话，简洁回复

直接发"写一个电池SOC安时积分法代码"，机器人直接回复完整代码。无需任何命令前缀。

笔记本一键运行最新代码 ✅ 无需魔法

在笔记本上操作：matlab.bat 启动前会自动 git pull。

① 安装 Git（首次）→ 克隆仓库到桌面

② 双击 matlab.bat → 自动拉取最新代码 → 启动 MATLAB

③ 在 Command Window 直接输入脚本名运行

代码通过 Claude Code 生成后自动 commit+push 到 GitHub，笔记本端 matlab.bat 启动时自动 git pull 拉取最新代码。

保持机器人长期运行 ⚡ 进阶操作

以上方式关掉终端后机器人就停止了。想让机器人 24 小时在线：

① 云服务器方案：买一台便宜的云服务器（腾讯云/阿里云最低配即可），把 tgbot.py 上传上去运行。

② 自家电脑方案（Windows）：使用 任务计划程序 设置开机自启 Python 脚本。

③ 自家电脑方案（Mac）：使用 nohup 命令：nohup python3 tgbot.py &

云服务器方案最稳定。腾讯云/阿里云最低配一个月约 10-30 元。注意云服务器上也需要安装 Python 和依赖包。

◆ 常见问题

Q：Claude Code 需要 Claude 账号吗？

不需要。Claude Code 是开源 Agent 框架，你可以给它接任何模型。本教程教的就是接 DeepSeek（国产模型），不需要 Claude 账号、不需要魔法。

Q：会被封号吗？需要外国手机号吗？

搭配国产模型使用时，不会封号，不需要外国手机号，不需要 Visa 卡。Claude Code 框架本身没有任何限制。

Q：装好了但 claude 命令找不到？

说明 Claude Code 没加到 PATH 环境变量。Mac 按安装完成后的提示执行 echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc 然后重启终端。Windows 重启终端即可。

Q：CC Switch 是什么？必须装吗？

CC Switch 是一个图形化工具，让你能方便地切换 Claude Code 后面接的模型。不是必须的——本教程的"方法一"直接改配置文件，完全不需要装任何额外软件。

Q：Telegram 机器人必须用自己的电脑跑吗？

不一定。你可以用云服务器跑，这样关掉自己电脑机器人也在线。也可以用自己的电脑测试，需要长期运行再迁移到服务器。

Q：DeepSeek API Key 怎么获取？要钱吗？

去 platform.deepseek.com 注册 → 控制台 → API Keys → 创建新 Key。DeepSeek 有免费额度，一般轻度使用足够了。
