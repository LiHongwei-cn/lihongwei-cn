0 Claude Code 是什么

Claude Code 是 Anthropic 的终端版 AI Agent 框架。不是网页聊天——它在终端里直接读/写文件、执行命令、管理项目，是真正能帮你干活的 AI。

🤖

Agent 框架

终端里帮你写代码、改文件、执行命令的智能助手

🔌

模型自由

可接入 Claude / DeepSeek / GLM 等任意模型

🛡️

国内可用

搭配国产模型不需要魔法、不需要外国手机号

📁

本地运行

代码在你电脑执行，不经过第三方服务器

核心概念：Agent 框架 = 身体，模型 = 脑子。Claude Code 是身体，你可以装国产模型的脑子（如 DeepSeek），效果依然很好。

1 安装 Claude Code

▎macOS 安装

打开终端 无需魔法

按 Command + 空格 搜索「终端」，打开终端 App。窗口末尾有 $ 符号在闪烁。

推荐 用 Homebrew 安装 无需魔法

如果没装过 Homebrew，先执行：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装完成后，安装 Claude Code：

brew install claude-code

备选：官方脚本安装 需要魔法

curl -fsSL https://claude.ai/install.sh | bash

看到 "Claude Code installed successfully" 即成功。

▎Windows 安装

先装 Git 无需魔法

Claude Code 在 Windows 依赖 Git Bash。按 Win + R 输入 wt 打开终端：

winget install Git.Git

推荐 用 WinGet 安装 无需魔法

winget install Anthropic.ClaudeCode

备选：官方脚本安装 需要魔法

irm https://claude.ai/install.ps1 | iex
安装后终端输入 claude --version 验证，输出版本号即成功。

2 接入 DeepSeek 模型（国产，有免费额度）

Claude Code 装好只是空壳，需要接模型。有 Claude 账号可以直接 claude 登录。这里教接 DeepSeek（国产），不需要 Claude 账号、不需要魔法。

准备工作：去 platform.deepseek.com 注册 → API Keys → 创建 Key → 复制保存（以 sk- 开头）。

▎方法一（推荐⭐）：直接配置 settings.json

找到配置文件 无需魔法

macOS：终端输入 open ~/.claude/settings.json

Windows：文件管理器地址栏输入 %USERPROFILE%\.claude\settings.json

如果文件不存在，在该目录下新建 settings.json。

写入配置

用文本编辑器打开，粘贴以下内容（替换 Key）：

{ "env": { "ANTHROPIC_BASE_URL": "https://api.deepseek.com/anthropic", "ANTHROPIC_AUTH_TOKEN": "sk-你的deepseek_api_key", "ANTHROPIC_MODEL": "deepseek-v4-pro", "ANTHROPIC_DEFAULT_HAIKU_MODEL": "deepseek-v4-pro", "ANTHROPIC_DEFAULT_SONNET_MODEL": "deepseek-v4-pro", "ANTHROPIC_DEFAULT_OPUS_MODEL": "deepseek-v4-pro" }, "theme": "dark" }
这六行把 Claude Code 内部所有模型都指向 DeepSeek，不管 Agent 请求哪个模型，实际走的都是 DeepSeek。

▎方法二（备选）：CC Switch 图形化工具

不想手动编辑配置文件的话，可以用 CC Switch 图形化管理模型（下载，需魔法）。

安装 CC Switch

macOS：

brew tap farion1231/ccswitch && brew install --cask cc-switch

Windows：下载安装包双击运行。

配置 DeepSeek

打开 CC Switch → Claude Code 栏点 "+" → 选 DeepSeek 或自定义 → 填入 API Key / Base URL / 模型名 → 添加。之后在 Claude Code 中输入 /model 即可切换。

4 部署全局规范（一键配置所有偏好）

这是用好 Claude Code 最关键的一步。CLAUDE.md + settings.json + rules + skills 一次性部署，让 Claude Code 从此自动记住你的偏好。

3.1 拉取规范仓库

Clone 到桌面

🍎 macOS

cd ~/Desktop && git clone https://github.com/LiHongwei-cn/lihongwei-cn.git

🪟 Windows

cd %USERPROFILE%\Desktop && git clone https://github.com/LiHongwei-cn/lihongwei-cn.git

进入 global-specs/ 目录，里面有 CLAUDE.md、settings.json、rules/、skills/、memory/。

3.2 配置文件映射（一张表说清楚）

源文件 (global-specs/)macOS 目标Windows 目标作用

CLAUDE.md~/.claude/CLAUDE.md%USERPROFILE%\.claude\CLAUDE.md全局用户指令（你是谁、行为准则）

3.3 极速版：一键部署（macOS）

已 clone 仓库并编辑好 settings.json 的 API Key 后，终端进入 global-specs/ 目录，复制粘贴以下全部：

cp settings.json ~/.claude/settings.json cp CLAUDE.md ~/.claude/CLAUDE.md cp settings.local.json ~/.claude/settings.local.json mkdir -p ~/.claude/rules ~/.claude/skills/neat-freak/references cp rules/*.md ~/.claude/rules/ && cp skills/neat-freak/*.md ~/.claude/skills/neat-freak/ && cp skills/neat-freak/references/*.md ~/.claude/skills/neat-freak/references/ cp project-CLAUDE.md ~/Desktop/lihongwei-cn/CLAUDE.md

3.4 极速版：一键部署（Windows PowerShell）

copy settings.json $env:USERPROFILE\.claude\settings.json copy CLAUDE.md $env:USERPROFILE\.claude\CLAUDE.md copy settings.local.json $env:USERPROFILE\.claude\settings.local.json mkdir $env:USERPROFILE\.claude\rules; copy rules\*.md $env:USERPROFILE\.claude\rules\ mkdir $env:USERPROFILE\.claude\skills\neat-freak\references; copy skills\neat-freak\SKILL.md $env:USERPROFILE\.claude\skills\neat-freak\; copy skills\neat-freak\references\*.md $env:USERPROFILE\.claude\skills\neat-freak\references\ copy project-CLAUDE.md $env:USERPROFILE\Desktop\lihongwei-cn\CLAUDE.md
部署后启动 Claude Code，输入 /neat 如果看到 skill 开始审查文件，说明全部部署成功。

5 搭建 Telegram AI 机器人

把 AI 接入 Telegram，发消息给机器人自动回复。代码生成 → 推送到 GitHub，论文报告 → 保存到 docs。

准备工作

① DeepSeek API Key — platform.deepseek.com 注册创建

② Telegram Bot Token — Telegram 搜索 @BotFather → 发送 /newbot → 按提示取名 → 获得 Token

③ Python 3.8+ — 终端输入 python3 --version 检查

安装依赖 + 运行

pip install python-telegram-bot openai python-dotenv

完整代码见仓库 bot/tgbot.py，基于 DeepSeek API 直接回复。设置环境变量 TELEGRAM_TOKEN 和 DEEPSEEK_API_KEY 后运行：

python bot/tgbot.py

保持长期运行

① 云服务器（最稳定）：腾讯云/阿里云最低配，24 小时在线

② macOS：nohup python3 tgbot.py &

③ Windows：任务计划程序设置开机自启

◆ 常见问题

Q：需要 Claude 账号吗？会被封吗？

不需要。搭配 DeepSeek 等国产模型，不需要 Claude 账号、不需要魔法、不需要外国手机号、不封号。

Q：装好了但 claude 命令找不到？

macOS：echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc 然后重启终端。Windows：重启终端即可。

Q：API 连接失败？

检查 settings.json 中 ANTHROPIC_AUTH_TOKEN 是否为有效 Key、ANTHROPIC_BASE_URL 是否为正确的 DeepSeek 地址。

Q：DeepSeek API Key 怎么获取？要钱吗？

去 platform.deepseek.com 注册 → API Keys → 创建 Key。新用户有免费额度，轻度使用足够。
