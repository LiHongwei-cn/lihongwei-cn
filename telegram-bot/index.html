1 工作原理

Telegram 消息 → Python Bot → DeepSeek API → AI 回复 → Telegram 聊天

用户在 Telegram 发消息 → Python 机器人接收 → 转发给 DeepSeek API → 拿到 AI 回复 → 发回 Telegram。
全程走 Telegram 和 DeepSeek 官方 API，不需要魔法、不需要外国手机号。

2 前置准备

Python 3.8+

终端输入 python3 --version 检查。如果没有：

macOS

brew install python@3.12

Windows

winget install Python.Python.3.12

或去 python.org 下载安装包。

DeepSeek API Key

去 platform.deepseek.com 注册 → API Keys → 创建 Key → 复制保存（sk- 开头）。新用户有免费额度。

Telegram Bot Token

Telegram 搜索 @BotFather → 发送 /newbot → 按提示取名字 → 获得 Token（格式：123456:ABC-DEF1234...）。

Bot 名字可以随便取，创建后 Token 请妥善保存，不要发给任何人。

3 部署运行

下载代码

从 GitHub 仓库获取 bot/ 目录：

git clone https://github.com/LiHongwei-cn/lihongwei-cn.git cd lihongwei-cn/bot

或直接在 GitHub 上下载 bot/ 文件夹的 ZIP。

安装依赖

pip install -r requirements.txt

依赖包：python-telegram-bot（Telegram API）、openai（DeepSeek API）、python-dotenv（环境变量）。

配置密钥

在 bot/ 目录下创建 .env 文件：

TELEGRAM_TOKEN=123456:ABC-DEF1234ghiJKL DEEPSEEK_API_KEY=sk-你的deepseek密钥
也可以用系统环境变量 export / setx 设置，启动脚本会自动读取。

启动机器人

macOS / Linux

bash start_bot.sh

Windows

start_bot.bat

看到 "Bot 已启动" 即可。去 Telegram 给你的 Bot 发消息试试。

4 保持 24 小时在线

方案一：云服务器 推荐

腾讯云/阿里云最低配（2核2G），月费几十块。代码上传后 nohup python3 tgbot.py & 即可 24 小时在线，最稳定。

方案二：macOS 后台运行

终端进入 bot/ 目录：

nohup python3 tgbot.py > bot.log 2>&1 &

Bot 在后台运行，关闭终端不影响。查看日志：tail -f bot.log

如需开机自启：系统设置 → 通用 → 登录项 → 添加 start_bot.sh。

方案三：Windows 开机自启

Win + R 输入 taskschd.msc → 创建基本任务 → 触发器选「计算机启动」→ 操作选 start_bot.bat。

也可以用仓库提供的 start_bot.vbs，双击即可无窗口后台运行。

6 主动推送通知

Bot 在运行时会自动保存聊天对象的 chat_id。之后可以从任何地方发通知到 Telegram：

python notify.py "任务完成：仿真结果已生成"

适合在脚本跑完后自动通知——在 MATLAB/Python 脚本末尾加一行 python notify.py "xxx已完成" 即可。

? 常见问题

启动后连接不上？

检查 .env 文件中的 TELEGRAM_TOKEN 和 DEEPSEEK_API_KEY 是否正确，Key 前后不要有多余空格或引号。在国内网络环境下 Telegram API 可能需要代理，确认代理已开启。

Bot 没反应 / 不回复？

终端看日志输出。常见原因：① Token 填错（去 @BotFather 重新获取）；② API Key 失效（去 DeepSeek 平台检查余额）；③ 网络不通（Telegram API 在国内被墙，需要开代理）。

提示 "Conflict: another bot instance is running"？

另一台设备/进程正在运行同一个 Bot。Bot 同一时间只能有一个实例在线。去其他设备停掉 Bot，或重启当前进程（新实例会自动踢掉旧实例）。

需要魔法吗？

Telegram 本身在国内需要代理才能连接，但 Bot 代码本身不需要额外的翻墙工具——只要你的服务器/电脑能访问 api.telegram.org 即可。DeepSeek API 国内直连，不需要魔法。

可以换其他 AI 模型吗？

可以。代码用的是 OpenAI 兼容接口，把 base_url 和 api_key 换成 GLM / Qwen / Moonshot 等任意兼容接口即可。模型名在 DEEPSEEK_MODEL 变量处修改。

怎么改机器人的回复风格？

编辑 tgbot.py 中的 SYSTEM_PROMPT 变量。当前是针对新能源汽车工程学生的提示词，你可以改成任何角色和风格。
