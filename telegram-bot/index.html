1 工作原理

Telegram 消息 → 蒙多 Bot → GitHub + SO 搜索 → DeepSeek-V4-Pro → 最优方案

普通消息 → DeepSeek 直接回复。
蒙多模式 → 并行搜索 GitHub 仓库 + Stack Overflow → 资料注入 AI → 给出对比分析后的最优方案。

2 部署指南

克隆仓库并安装依赖

git clone https://github.com/LiHongwei-cn/lihongwei-cn.git cd lihongwei-cn/bot pip install -r requirements.txt

配置环境变量

复制模板并填入你的密钥：

cp .env.example .env # 编辑 .env，填入： # TELEGRAM_TOKEN=你的Token（从 @BotFather 获取） # DEEPSEEK_API_KEY=sk-你的Key（从 platform.deepseek.com 获取）

启动 Bot

# macOS / Linux bash start_bot.sh # Windows start_bot.bat # 或直接运行 python tgbot.py

看到 Bot 已启动 即成功。去 Telegram 发消息测试。

开机自启（可选）

# macOS — LaunchAgent 开机自启 bash setup_login_start.sh # Windows — 开机自启快捷方式 # 双击 setup_startup.vbs
Bot 支持 macOS + Windows + Linux，所有启动脚本跨平台可用。

3 蒙多实战示例

/mundo React 列表渲染 1000 条卡顿

蒙多搜索 react-window、react-virtualized、TanStack Virtual，四维对比后推荐 react-window + React.memo 方案。

直接说「蒙多帮我看看这个报错」
