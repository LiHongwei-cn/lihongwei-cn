免费 VPN 自建指南

基于 Cloudflare 免费套餐，零成本搭建属于你自己的 VPN 节点。
参考 YouTube 博主 零度解说 的 2026 年最新教程整理。

⚠️ 本教程仅用于学习和研究网络技术，请遵守当地法律法规，勿用于非法用途。
✅ 无需梯子 可直接访问 ⚠️ 需要梯子 需自备网络环境 ⚡ 完成后可用 搭建成功后可访问

A 方法一：Cloudflare Zero Trust + WARP（推荐）

利用 Cloudflare 官方的 Zero Trust 平台 + WARP 客户端，免费、稳定、速度快。适合不想折腾代码、希望即开即用的用户。
关键优势：Cloudflare 网站 国内可直接访问，整个搭建过程 不需要梯子。

注册 Cloudflare 账号 ✅ 无需梯子

打开 dash.cloudflare.com/sign-up，用邮箱注册一个免费账号。

Cloudflare 在国内有服务器，注册页面可直接打开。

进入 Zero Trust 仪表盘 ✅ 无需梯子

登录后进入 one.dash.cloudflare.com。

① 输入一个 团队名称（随便写，如你的名字拼音）

② 选择 Free 免费套餐（最多支持 50 个席位）

创建设备注册策略 ✅ 无需梯子

Settings → WARP Client → Device enrollment permissions → Manage

添加规则：Anyone with an email → 输入你的邮箱 → 保存。

下载 WARP 客户端 ✅ 无需梯子

官方下载页（国内可打开）：
developers.cloudflare.com/cloudflare-one/connections/connect-devices/warp/download-warp/

支持 Windows / macOS / Linux / Android / iOS。

如果官方下载慢，可在百度搜索 "WARP 客户端 下载" 找国内镜像。

登录 WARP 客户端 ✅ 无需梯子

打开 WARP → 设置 → 账户 → 登录，选择 Cloudflare Zero Trust。

浏览器会自动打开，输入 团队名称，用邮箱登录。

切换为 MASQUE 协议 ✅ 无需梯子

解决运营商封锁，速度比默认 WireGuard 快约 1 倍。

Zero Trust 后台 → Settings → WARP Client → Profile settings → Default：

找到 Transport protocol，从 WireGuard 改为 MASQUE → 保存。

MASQUE 基于 HTTP/3 (QUIC)，速度快且不容易被识别封锁，实测可跑满千兆带宽。

开始使用 WARP ⚡ 搭建成功即可用

客户端开关切换到 已连接，此时你的网络已通过 Cloudflare 全球网络转发。

以后每次使用只需打开 WARP 客户端连接即可。

手机端 iOS/Android 需用 WireGuard 协议（不支持 MASQUE），下载 1.1.1.1 + WARP 或 Cloudflare One Agent。

B 方法二：Cloudflare Pages + edgetunnel

利用 Cloudflare Pages 部署 VLESS 代理服务，配合 Clash 客户端使用。适合想要自定义节点、使用 Clash 分流管理的高级用户。
注意：Fork GitHub 项目这一步需要梯子，后续操作 Cloudflare 后台则不需要。

注册 Cloudflare 账号 ✅ 无需梯子

打开 dash.cloudflare.com/sign-up 注册免费账号。

Fork edgetunnel 项目 ⚠️ 需要梯子

打开 github.com/cmliu/edgetunnel（GitHub 需梯子才能访问），点击 Fork 复制到你的账号下。

如果暂时没梯子，可以找人帮你 Fork 后把仓库权限转给你，或者用 gitclone.com 加速。项目名不要用 "edgetunnel"。

生成 UUID ✅ 无需梯子

UUID 是你的节点身份标识。打开 www.uuidgenerator.net 生成一个，复制保存。

创建 Pages 项目 ✅ 无需梯子

Cloudflare 后台 → Workers 和 Pages → Pages → 创建 → 连接到 Git。

授权 GitHub，选择 Fork 的仓库 → 保存并部署（首次会失败，正常）。

配置环境变量 ✅ 无需梯子

Pages 项目 → 设置 → 变量和机密 → 添加变量：

变量名说明是否必填示例值
UUID节点标识✅ 必填你生成的 UUID
PROXYIP代理转发地址推荐proxyip.cmliussss.net
ADD优选 IP 列表可选[2400:cb00::...]

添加后返回部署页面点击 重试部署。

获取订阅链接 ✅ 无需梯子

部署成功后访问：

https://你的项目名.pages.dev/你的UUID

页面返回 VLESS 节点链接和 Clash 订阅地址。

导入 Clash 使用 ⚠️ 需要梯子才能测通

复制订阅地址 → 打开 Clash Verge Rev → 订阅 → 新建 → 粘贴 URL → 导入 → 启用。

导入后节点是否能连通，取决于你的网络环境和 PROXYIP 配置。

Cloudflare 禁止使用 Workers/Pages 部署代理，存在封号风险。仅供学习实验。

C Clash 客户端下载与使用指南

Clash 是最主流的代理客户端，支持规则分流、自动选节点、订阅更新。下载 Clash 客户端本身需要梯子（GitHub 被封锁）。

客户端平台下载地址需梯子
Clash Verge Rev Windows / macOS / Linux GitHub Releases ⚠️
FlClash Android / Win / Mac / Linux GitHub Releases ⚠️
ClashX Pro macOS 专用 GitHub Releases ⚠️
客户端下载不了？安装好方法一的 WARP 后再下载，或者找国内网盘搬运版（百度搜索 "Clash Verge Rev 下载"）。

Clash Verge Rev 快速上手

下载安装

Windows 选 Clash.Verge_Windows_x64-setup.exe

macOS Intel 选 _x64.dmg，M 芯片选 _aarch64.dmg

添加订阅

打开 Clash Verge Rev → 订阅 → 新建 → 粘贴 URL → 导入 → 启用。

订阅地址从哪里来？方法二搭建成功后获取，或使用其他机场订阅链接。

开启代理

设置 → 打开 系统代理。或使用 TUN 模式（所有流量自动接管）。

选模式

Rule（规则）：国内直连，国外走代理（推荐日常）

Global（全局）：所有流量走代理

Direct（直连）：所有流量不走代理

更新订阅

点击 刷新 手动更新。可在 设置 → 订阅设置 开启自动更新。
