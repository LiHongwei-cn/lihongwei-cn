免费 VPN 自建指南

基于 Cloudflare 免费套餐，零成本搭建属于你自己的 VPN 节点。
参考 YouTube 博主 零度解说 的 2026 年最新教程整理。

⚠️ 本教程仅用于学习和研究网络技术，请遵守当地法律法规，勿用于非法用途。
✅ 无需梯子 可直接访问 ⚠️ 需要梯子 需自备网络环境 ⚡ 完成后可用 搭建成功后可访问

A 方法一：Cloudflare Zero Trust + WARP（推荐）

利用 Cloudflare 官方的 Zero Trust 平台 + WARP 客户端，免费、稳定、速度快。适合不想折腾代码、希望即开即用的用户。
关键优势：Cloudflare 网站 国内可直接访问，整个搭建过程 不需要梯子。

注册 Cloudflare 账号 ✅ 无需梯子

打开浏览器 → 地址栏输入 dash.cloudflare.com/sign-up 并回车。

页面正中间会出现注册表单，依次填写：

① Email 输入框：填你的邮箱地址

② Password 输入框：设一个密码（至少 8 位，含大小写字母）

③ 点击下方蓝色的 "Create Account" 按钮

④ 去邮箱收件箱找到 Cloudflare 发的验证邮件，点击其中的 "Verify email" 按钮

⑤ 验证成功后自动跳转回 Cloudflare 后台首页（显示 "Welcome to Cloudflare"）

Cloudflare 在国内有服务器，注册页可直接打开。如果显示英文界面，左下角有个地球图标可以切换语言。

进入 Zero Trust 仪表盘 ✅ 无需梯子

在 Cloudflare 后台首页，找到左侧蓝色的 "Zero Trust" 图标（像个盾牌）并点击。

如果找不到：直接在浏览器地址栏打开 one.dash.cloudflare.com。

首次进入会看到"Onboard to Cloudflare Zero Trust"页面：

① 在页面正中间的输入框中，输入一个 团队名称（随意，比如你的名字拼音，记下来后面要用）

② 页面下方会出现三个套餐卡片，点击 "Free" 那张卡片（上面写着 "Free $0/seat"）

③ 点击右下角的蓝色 "Continue" 按钮

④ 弹出一个确认弹窗，核对套餐无误后点击 "Proceed"

⑤ 等待几秒，自动进入 Zero Trust 后台主页

创建设备注册策略 ✅ 无需梯子

进入 Zero Trust 后台后，看左侧深色导航栏：

① 点击 "Settings"（齿轮图标，在导航栏靠下方）

② 在右侧页面中，点击顶部导航的 "WARP Client" 标签页

③ 往下翻找到 "Device enrollment permissions" 一栏，点击右侧的蓝色 "Manage" 按钮

④ 会打开一个规则编辑页面，点击 "Add a rule" 按钮

⑤ 规则名称随便填（如 "allow all"），规则条件设置：

- 下拉框选 "Anyone with an email"

- 输入框填你的邮箱地址

⑥ 点击右下角的蓝色 "Save" 按钮

⑦ 页面提示 "Success" 表示创建成功

下载 WARP 客户端 ✅ 无需梯子

打开新标签页，访问官方下载页：
developers.cloudflare.com/cloudflare-one/connections/connect-devices/warp/download-warp/

页面往下翻，会看到一个操作系统列表表格：

① Windows：点击表格中 "Windows" 那行的下载链接（下载 .msi 安装包）

② macOS：对应行下载 .pkg 安装包

③ Android / iOS：去应用商店分别搜索 "1.1.1.1 + WARP" 或 "Cloudflare One Agent"

下载完成后，双击安装包按提示安装（一路点"下一步"即可）。

如果官方下载慢，可在百度搜索 "WARP 客户端 下载" 找国内网盘搬运版。

登录 WARP 客户端 ✅ 无需梯子

安装完成后启动 WARP：

① 在桌面右下角任务栏找到 WARP 图标（蓝色圆形小图标）→ 右键 → 点击 "Preferences"（首选项）

② 或者双击任务栏图标打开 WARP 窗口 → 点击右上角的 齿轮图标（Settings）

③ 在设置页面点击 "Account"（账户）选项卡

④ 点击 "Login with Cloudflare Zero Trust" 按钮（蓝色）

⑤ 浏览器会自动打开一个页面，输入你之前设的 团队名称 → 点击 "Continue"

⑥ 浏览器跳转到 Cloudflare 登录页面，输入你的邮箱 → 获取验证码 → 登录

⑦ 登录成功后会提示 "You are now logged in"，可以关掉浏览器回到 WARP 窗口

切换为 MASQUE 协议 ✅ 无需梯子

回到浏览器，进入 Zero Trust 后台：

① 左侧导航栏点击 "Settings"（齿轮图标）

② 顶部标签页点 "WARP Client"

③ 找到 "Profile settings" 一栏 → 点击 "Default" 右侧的 齿轮图标（编辑按钮）

④ 在弹出的编辑页面中，找到 "Transport protocol"（传输协议）下拉框

⑤ 默认是 "WireGuard"，改成 "MASQUE"

⑥ 点击页面下方的蓝色 "Save" 按钮

⑦ 提示保存成功后关闭页面

MASQUE 基于 HTTP/3 (QUIC)，速度比默认 WireGuard 约快 1 倍，且不容易被运营商识别封锁，实测可跑满千兆带宽。

开始使用 WARP ⚡ 搭建成功即可用

回到 WARP 桌面客户端主窗口：

① 你会看到页面正中间有一个大大的开关按钮，上面写着 "Disconnected"（未连接）

② 点击这个开关，它会开始切换，几秒后变成 "Connected"（已连接），按钮变成绿色

③ 此时你的网络已经通过 Cloudflare 全球网络转发，可以访问被屏蔽的网站了

④ 以后每次使用：打开 WARP 客户端 → 点击开关连接即可

手机端 iOS/Android 不支持 MASQUE 协议，下载 "1.1.1.1 + WARP" App → 设置 → 账户 → 同样用 Zero Trust 登录即可（手机端走 WireGuard 协议，速度会比桌面端慢一点）。

B 方法二：Cloudflare Pages + edgetunnel

利用 Cloudflare Pages 部署 VLESS 代理服务，配合 Clash 客户端使用。适合想要自定义节点、使用 Clash 分流管理的高级用户。
注意：Fork GitHub 项目这一步需要梯子，后续操作 Cloudflare 后台则不需要。

注册 Cloudflare 账号 ✅ 无需梯子

打开 dash.cloudflare.com/sign-up 注册免费账号。

Fork edgetunnel 项目 ⚠️ 需要梯子

打开 github.com/cmliu/edgetunnel（GitHub 需梯子才能访问），点击 Fork 复制到你的账号下。

如果暂时没梯子，可以找人帮你 Fork 后把仓库权限转给你，或者用 gitclone.com 加速。项目名不要用 "edgetunnel"。

生成 UUID ✅ 无需梯子

UUID 是你的节点身份标识。打开 www.uuidgenerator.net 生成一个，复制保存。

创建 Pages 项目 ✅ 无需梯子

Cloudflare 后台 → Workers 和 Pages → Pages → 创建 → 连接到 Git。

授权 GitHub，选择 Fork 的仓库 → 保存并部署（首次会失败，正常）。

配置环境变量 ✅ 无需梯子

Pages 项目 → 设置 → 变量和机密 → 添加变量：

变量名说明是否必填示例值
UUID节点标识✅ 必填你生成的 UUID
PROXYIP代理转发地址推荐proxyip.cmliussss.net
ADD优选 IP 列表可选[2400:cb00::...]

添加后返回部署页面点击 重试部署。

获取订阅链接 ✅ 无需梯子

部署成功后访问：

https://你的项目名.pages.dev/你的UUID

页面返回 VLESS 节点链接和 Clash 订阅地址。

导入 Clash 使用 ⚠️ 需要梯子才能测通

复制订阅地址 → 打开 Clash Verge Rev → 订阅 → 新建 → 粘贴 URL → 导入 → 启用。

导入后节点是否能连通，取决于你的网络环境和 PROXYIP 配置。

Cloudflare 禁止使用 Workers/Pages 部署代理，存在封号风险。仅供学习实验。

Cloudflare 后台dash.cloudflare.com ✅
edgetunnel 仓库github.com/cmliu/edgetunnel ⚠️

C Clash 客户端下载与使用指南

Clash 是最主流的代理客户端，支持规则分流、自动选节点、订阅更新。下载 Clash 客户端本身需要梯子（GitHub 被封锁）。

客户端平台下载地址需梯子
Clash Verge Rev Windows / macOS / Linux GitHub Releases ⚠️
FlClash Android / Win / Mac / Linux GitHub Releases ⚠️
ClashX Pro macOS 专用 GitHub Releases ⚠️
客户端下载不了？安装好方法一的 WARP 后再下载，或者找国内网盘搬运版（百度搜索 "Clash Verge Rev 下载"）。

Clash Verge Rev 快速上手

下载安装

Windows 选 Clash.Verge_Windows_x64-setup.exe

macOS Intel 选 _x64.dmg，M 芯片选 _aarch64.dmg

添加订阅

打开 Clash Verge Rev → 订阅 → 新建 → 粘贴 URL → 导入 → 启用。

订阅地址从哪里来？方法二搭建成功后获取，或使用其他机场订阅链接。

开启代理

设置 → 打开 系统代理。或使用 TUN 模式（所有流量自动接管）。

选模式

Rule（规则）：国内直连，国外走代理（推荐日常）

Global（全局）：所有流量走代理

Direct（直连）：所有流量不走代理

更新订阅

点击 刷新 手动更新。可在 设置 → 订阅设置 开启自动更新。

将下载的 APK 传到手机 → 点击安装 → 允许"安装未知来源应用"。

导入订阅链接 ✅ 无需梯子

① 打开 App，点击底部 "配置"（Profiles）

② 点击右上角 ➕ → 选择 "从 URL 导入"

③ 填写名称（随意），在 URL 处粘贴你的订阅地址

④ 保存 → 回到配置页选中该配置

启动代理 ✅ 无需梯子

回到首页 → 点击 "已停止" → 启动。

系统会提示"VPN 连接请求"，点击 确定。

状态变为绿色即表示运行中。

进阶设置（推荐）

① 网络 → DNS 劫持 → 关闭（避免与系统 DNS 冲突）

② Meta 特性 → 嗅探设置 → 不修改

③ 开启 自动更新订阅（保持节点最新）

E 苹果手机：Shadowrocket（小火箭）配置

iOS 端最主流的代理客户端，支持 SS/SSR/Vmess/VLESS/Trojan 等多种协议。此应用需要 美区 Apple ID 才能下载（售价 $2.99）。
下方先教你怎么创建美区 ID。

▎前置准备：注册美区 Apple ID

以下方法 2026 年实测有效，不需要美国信用卡，用国内手机号即可完成。

准备工作 ✅ 无需梯子

邮箱：注册一个新邮箱（推荐 Gmail/Outlook，不要用 QQ/163 邮箱，易触发风控）

手机号：国内 +86 手机号（确保能收国际短信）

地址：准备一个美国免税州地址（俄勒冈 Oregon / 特拉华 Delaware / 阿拉斯加 Alaska）

网络：建议用手机自带流量 + Safari 浏览器，关闭代理

为什么要用免税州地址？美国部分州购买数字产品要交消费税，俄勒冈等州没有消费税，选地址时选这些州可省税。

在苹果官网创建账号 ✅ 无需梯子

用手机 Safari 打开 account.apple.com

点击 "创建你的 Apple 账户"，按提示填写：

① 国家/地区：一定要选 「United States」

② 姓名：填拼音（如 Xiaoming Wang）

③ 出生日期：必须大于 18 岁

④ 手机号：选中国（+86），填你的手机号

⑤ 密码：至少 8 位，含大小写+数字，不能含姓名/生日/邮箱前缀

依次输入邮箱验证码 + 手机短信验证码，完成注册。

登录 App Store 激活（最关键） ✅ 无需梯子

⚠️ 绝对不要在「设置」里登录 iCloud！

① 打开 App Store → 点右上角头像 → 退出当前账号

② 用新注册的美区账号登录

③ 搜索一个 免费 App（如 YouTube、TikTok、Instagram）

④ 点击「获取」，系统会弹出 "检查" 提示

⑤ 付款方式选「None/无」（关键！如果有信用卡选项说明国家没选对）

⑥ 填写地址：填写你准备的 免税州地址，电话填 000-000-0000

⑦ 提交 → 激活成功！

付款方式没有「None」选项？说明注册时国家没选美国，需要重新注册。关闭代理 + 用蜂窝数据 + Safari 无痕模式重试。

验证可用

激活后搜索 Shadowrocket，如果能看到此应用（售价 $2.99），说明美区 ID 已生效。

以后每次下载美区 App，App Store 会提示你输入此账号的密码。

美区账号安全铁律：
① 此账号仅在 App Store 登录，绝不登录 iCloud（否则可能被锁机）
② 需要付费买 App 时，去 亚马逊美国官网 或 苹果美国官网 买电子礼品卡充值，不要在淘宝买（黑卡会封号）
③ 切回国内 App Store 后，之前用美区下的 App 依然能正常更新

▎Shadowrocket 配置使用

购买并下载 Shadowrocket ✅ 美区 ID 登录后

美区 App Store 搜索 Shadowrocket → 售价 $2.99 → 购买下载。

没有美区礼品卡？跳过购买，改用 Stash（国区可下）或 Sing-box（免费开源）替代。

导入订阅 ✅ 无需梯子

① 打开 Shadowrocket → 底部 "设置" → 进入 "服务器订阅"

② 打开 "打开时更新" 开关

③ 返回首页 → 点右上角 ➕ → 类型改为 Subscribe

④ URL 处粘贴你的订阅链接 → "备注" 填名称 → 保存

⑤ App 自动更新服务器列表

开启代理 ✅ 无需梯子

① 首页选择一个节点（看延迟值，越低越好）

② 打开顶部 "连接" 开关

③ 首次会提示"添加 VPN 配置" → 点 允许 → 验证 Face ID

设置分流规则

首页顶部 "全局路由" 选 "配置" 模式：

Rule（配置）：国内直连，国外代理（推荐日常）

Proxy（代理）：全部走代理

Direct（直连）：全部不走代理

其他建议

延迟测试：设置 → 延迟测试方法 → 改为 CONNECT（更准确）

测速选节点：首页点 "连通性测试" 测速，选延迟低、速度快的节点

更新订阅：下拉首页即自动刷新节点
