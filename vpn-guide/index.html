免费 VPN 自建指南

基于 Cloudflare 免费套餐，零成本搭建属于你自己的 VPN 节点。
参考 YouTube 博主 零度解说 的 2026 年最新教程整理。

⚠️ 本教程仅用于学习和研究网络技术，请遵守当地法律法规，勿用于非法用途。

A 方法一：Cloudflare Zero Trust + WARP（推荐）

利用 Cloudflare 官方的 Zero Trust 平台 + WARP 客户端，免费、稳定、速度快（实测支持千兆宽带）。适合不想折腾代码、希望即开即用的用户。

注册 Cloudflare 账号

打开 dash.cloudflare.com/sign-up，用邮箱注册一个免费账号。

进入 Zero Trust 仪表盘

登录后进入 one.dash.cloudflare.com。

根据提示：

① 输入一个 团队名称（随便写，如你的名字拼音）

② 选择 Free 免费套餐（最多支持 50 个席位，完全够用）

创建 WARP 设备注册策略

进入 Settings → WARP Client → Device enrollment permissions，点击 Manage。

添加一条规则：Anyone with an email → 输入你的邮箱地址 → 保存。

下载并安装 WARP 客户端

从 Cloudflare 官方下载 WARP 客户端：

developers.cloudflare.com/cloudflare-one/connections/connect-devices/warp/download-warp/

支持 Windows / macOS / Linux / Android / iOS。

登录 WARP 客户端

安装后打开 WARP 客户端 → 设置 → 账户 → 登录，选择 Cloudflare Zero Trust。

浏览器会自动打开，输入你刚才设置的 团队名称，然后用邮箱登录。

切换为 MASQUE 协议（关键步骤）

这一步解决了运营商封锁 WARP 的问题，速度也比默认的 WireGuard 快约 1 倍。

在 Zero Trust 后台 → Settings → WARP Client → Profile settings → Default：

① 找到 Transport protocol，从 WireGuard 改为 MASQUE

② 保存后客户端会自动重新连接

MASQUE 协议是 Cloudflare 最新的传输协议，基于 HTTP/3 (QUIC)，速度快且不容易被识别封锁。实测可跑满千兆带宽。

使用 WARP

客户端连接成功后，开关切换到 已连接 状态即可。

如果只需访问特定网站（分流通代理），可以在 WARP 客户端设置 Split Tunnels（分离隧道）模式。

手机端（iOS/Android）需要使用 WireGuard 协议（不支持 MASQUE），下载 1.1.1.1 + WARP 或 Cloudflare One Agent 即可。

B 方法二：Cloudflare Pages + edgetunnel（进阶）

利用 Cloudflare Pages 部署 VLESS 代理服务，配合 Clash 等客户端使用。适合希望自定义节点、使用 Clash 分流管理的用户。

注册 Cloudflare 账号

打开 dash.cloudflare.com/sign-up 注册免费账号。

Fork edgetunnel 项目

打开 github.com/cmliu/edgetunnel，点击右上角 Fork，将项目复制到你的 GitHub 账号下。

项目名称不要用 "edgetunnel"，改为其他任意名称。

生成 UUID

UUID 是你的节点身份标识。打开以下网址生成一个：

www.uuidgenerator.net

点击 "Generate"，复制生成的 UUID 字符串。

创建 Pages 项目

进入 Cloudflare 后台 → Workers 和 Pages → Pages → 创建 → 连接到 Git。

选择你 Fork 的 edgetunnel 仓库 → 保存并部署。

首次部署会失败，正常现象。

配置环境变量

进入 Pages 项目 → 设置 → 变量和机密 → 添加变量：

变量名说明示例值
UUID你的节点标识（必填）你生成的 UUID
PROXYIP代理转发地址（推荐）proxyip.cmliussss.net
ADD优选 IP 列表（可选）[2400:cb00::...]
ADMIN管理面板密码（可选）任意密码

添加完成后，返回部署页面点击 重试部署。

获取订阅链接

部署成功后，访问：

https://你的项目名.pages.dev/你的UUID

页面会返回 VLESS 节点链接和 Clash 订阅地址。

导入客户端使用

复制 Clash 订阅地址，在 Clash Verge Rev 或 FlClash 中导入即可使用。

如需自定义域名，在 Pages 设置中添加已托管在 Cloudflare 的域名即可。
Cloudflare 明确禁止使用 Workers/Pages 部署代理服务，存在封号风险。建议仅用于学习和实验，不要大规模传播节点信息。

Clash Verge Rev 快速上手

下载安装

根据你的系统下载对应版本：

Windows：Clash.Verge_Windows_版本号_x64-setup.exe

macOS Intel：Clash.Verge_macOS_版本号_x64.dmg

macOS M1/M2/M3：Clash.Verge_macOS_版本号_aarch64.dmg

添加订阅

打开 Clash Verge Rev → 订阅 → 新建：

① 在 URL 一栏粘贴你的订阅链接

② 点击 导入

③ 点击 启用 开关

开启系统代理

在 设置 页面打开 系统代理 开关。

或者使用 TUN 模式（虚拟网卡模式，所有流量自动接管，无需配置浏览器）。

选择节点模式

Clash 有三种模式：

Rule（规则）：国内流量直连，国外走代理（推荐日常使用）

Global（全局）：所有流量都走代理

Direct（直连）：所有流量都不走代理

日常使用选 Rule 模式即可。访问国内网站不受影响，访问国外网站自动走代理。

更新订阅

订阅链接失效或节点更新时，点击订阅旁的 刷新 按钮即可手动更新。

也可在 设置 → 订阅设置 中开启 自动更新。
